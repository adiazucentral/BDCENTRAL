Estadísticas
El módulo de estadística permite a los usuarios generar informes estadísticos para un análisis inmediato que le servirán de herramientas al laboratorio clínico para la  toma de decisiones.

A continuación, se muestra una ventana típica con sus partes:

 

1. Rango de fecha en la que el sistema realizara la busqueda de ordenes, si el tipo de reporte es mensual, este rango podra ser de maximo un año

2. Campo para seleccionar por que tipo de fecha desea relizar la busqueda (fecha de ingreso de la prueba o fecha de validacion de los resultados).

3. Tipos de reporte que pueden ser generados por el sistema, las diferentes opciones son las siguientes:     

  •  Condensado: Genera un informe con los conteos de las agrupaciones seleccionadas, un ejemplo del informe generado es el siguiente:

  •  Detallado: Genera un informe con los conteos de las agrupaciones seleccionadas y adiciona un detalle de las pruebas que pertenecen a cada grupo, un ejemplo del informe generado es el siguiente:

 

  •  Grafica: Genera un informe con graficas representativas de los datos seleccionados en el primer grupo, los demas grupos solo filtran el listado de ordenes. un ejemplo del informe generado es el siguiente:

 Mensual: Genera un informe con conteos mensuales de ordenes   que correspondan a los datos seleccionados en el primer grupo, los demas grupos solo filtran el listado de ordenes, un ejemplo del informe generado es el siguiente:

4.Filtro por el estado(ordenadas, validadas, impresas) de las pruebas que van a aplicar para el conteo que realiza el sistema.

5. Filtro por tipo de orden.

6. Filtro para el tipo de agrupacion, si se selecciona demograficos, se cargaran todos los demográficos fijos y codificados que tenga configurado el sistema. si se selecciona pruebas el siguiente nivel de agrupacion no podra ser por demográficos.

7. Lista de selección de para los demográficos codificados del sistema o campos de las pruebas  (áreas, nivel, laboratorio,pruebas, grupos etarios), si en este listado se selecciona la opción de pruebas no se podra seleccionar ningun item en el siguiente nivel de agrupación.

8. Botón para seleccionar todos los items de la lista.

9. Icono informativo que representa los items que van a ser exluidos del filtro y agrupación.

10. Listado de los item que va a ser exluidos o que van a aplicar al filtro y la agrupación esta opcion depende del boton para la selección de todos los items.

11. Icono informativo que representa los items que van a aplicar al filtro y a la agrupación

12. Niveles de agrupación y filtros para las ordenes.

13. Botón para generar el informe con las diferentes opciones escogidas. 

Nota: el conteo de las pruebas se vera afectado por el valor multiplicar por que tiene configurado cada prueba
