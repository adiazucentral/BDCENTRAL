8. Botonera de turnos.
En el panel inferior del formulario de Ingreso de órdenes que corresponde a la botonera, aparecen otros botones a la derecha referentes al funcionamiento de turnos.

En la imagen anterior se observa la secuencia a seguir para comenzar la funcionalidad de turnos. El usuario selecciona la opción Ingreso de órdenes, selecciona una taquilla y pulsa el botón Iniciar. Aparecen los siguentes botones:

8.1. Botón Turno manual.
Al pulsar este botón se abre una ventana Llamada de turno. Aparece una lista de turnos disponibles para llamado, contiene el código del turno, el servicio, el tipo de turno, la prioridad, el tiempo de espera y un botón al final que se acciona para gestionar el turno.

En la anterior imagen el usuario gestiona el turno y se despliega otra ventana:


                                                          [Imagen de referencia - Turno 01]

Del lado izquierdo aparece en grande el código del turno con el fondo del color correspondiente al botón pulsado en la lista de turnos y del lado derecho los demás datos. En la parte inferior aparece una botonera que tiene varias funciones.

 Atender: Este botón habilita los demás botones para comenzar a gestionar los turnos.
 Cancelar: Este botón abre una ventana de confirmación donde el usuario puede seleccionar un motivo de cancelación del turno.
       
 Reservar: Este botón abre una ventana de confirmación advirtiendo que el turno no podrá ser llamado por otro usuario. Al confirmar hace una reserva del turno y cuando se quiere seleccionar ese mismo turno, aparece un mensaje amarillo indicando el blooqueo.
       
 Aplazar: Este botón despliega una ventana para posponer un turno en curso, el usuario tiene la posibilidad de digitar el tiempo a posponer y el motivo por el cual se aplaza el turno.
       
 Asociar a la orden: Este botón despliega una ventana que busca por número de historia las órdenes activas correspondientes para asociarles un turno.

Una vez que se ejecute la acción en la parte superior del panel de Orden aparece el turno relacionado.
  
 Ingresar a la orden: Este botón cierra la ventana y deja listo el turno para asociarse a una nueva orden ingresada. Los botones Transferencia y Terminar turno quedan automáticamente habilitados.

8.2. Botón Turno automático.
 Al pulsar este botón se abre de una vez la ventana correspondiente a la gestión de un turno consecutivo como se muestra en la imagen de referencia Turno 01. A partir de ahí se ejecutan los mismos pasos anteriormente nombrados.

8.3. Botón Transferencia.
 Este botón es el encargado de transferir el turno a un servicio, dicho turno solo se podrá transferir a un servicio.

8.4. Botón Terminar turno.
 El botón muestra un mensaje de confirmación indicando si el usuario desea terminar el turno en curso.

 

Luego de terminar el turno, la botonera quedan como en su estado inicial cuando aparecen. Esta terminación de turno es distinta a la que se observa en la primera imagen.  El botón rojo termina la atención de todos los turnos y desaparece la botonera correspondiente. La gestión de turnos también podrá ser pausada cuando esté en ejecución.

Recordemos que el usuario inicialmente debe seleccionar una taquilla para iniciar la gestión de turnos (Gestionando turnos). Una vez terminada dicha gestión, se debe liberar la taquilla con el usuario en línea.
