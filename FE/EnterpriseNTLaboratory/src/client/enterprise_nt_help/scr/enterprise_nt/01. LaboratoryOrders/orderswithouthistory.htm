Creación de ordenes sin historia

En esta ventana es donde se gestionan la creación de órdenes que no tengan dependencia de ninguna historia clínica. El formulario solo tiene la capacidad de crear órdenes, no de editarlas. Se pueden crear una o varías órdenes en bloque y el número de orden dependerá de la llave de configuración. (Orden manual u orden automática).

A continuación, se muestra la ventana correspondiente para la asignación de historias.

1. Seleccionar.
En este bloque se muestran las dos opciones para crear órdenes sin historia; se selecciona por Orden donde se almacena una sola orden y en el caso de que la llave de configuración de orden automática esté desactivada el cuadro de texto del bloque de Orden se habilita, o por Rango donde aparece el número de órdenes que serán introducidas y en caso de que la llave de configuración de orden automática esté desactivada, aparece un cuadro de texto donde se deberá digitar la orden inicial.

2. Bloque de Orden.
Es el contenedor de los controles para gestionar cada campo de una orden. El cuadro de orden se habilita cuando la llave de configuración de orden automática esté desactivada. Los campos con asterisco (*) son obligatorios.

3. Cantidad de órdenes.
Pueden ser manuales o automáticas. Aquí se digita o se puntea el número de órdenes que se desean crear, aparece solo cuando se selecciona la opción Rango.

4. Orden inicial.
Es un campo numérico donde se digita el número de la orden.

5. Buscar pruebas.
En este control se buscan las pruebas por código y/o nombre para agregar a la lista. Si se desea quitar una prueba de la lista, se debe digitar el signo (-) seguido del código de la prueba. Este control especial sirve para desplegar una lista (Auto-completar) y se puede visualizar si la prueba es una perfil, paquete o simplemente una prueba.

6. Lista de pruebas.
Es una grilla que consta de unas columnas: Código, Nombre, Tarifa, Precio, posee un pie de grilla que visualiza el total del precios de las pruebas agregadas. La tarifa se puede seleccionar y con ello se cambia el precio de cada prueba. Tanto la tarifa y el precio se visualizan siempre y cuando la lleve de configuración de Tarifa esté activa en el maestro de instalación.

7. Nuevo.
Es el botón para limpiar y habilitar todos los controles del formulario y dejarlo dispuesto para gestionar nuevas órdenes sin historia.

8. Guardar.
Es el botón para almacenar una o varias órdenes sin historia.
