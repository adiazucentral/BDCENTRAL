Herramientas de Impresión

las herramientas de Impresión es una herramienta que provee la aplicación para configurar las impresora que utilizara el usuario. 

A continuación, se muestra una ventana típica con sus partes: 


[Imagen de herramientas de impresiòn]

1. Serial de impresión.   
Se escribe el serial de la impresora configurada, el cual se obtiene cuando se configura el programa del cliente de impresiòn. 

2. Serial de impresión.   
Al dar click en el botòn se descarga un archivo comprimido, debe tener un programa instalado para descomprimirlo. 

Abra la ubicaciòn del archivo que se descargo, que por lo general se encuentra en descargas.

.

Cree una carpeta, en mi caso la pondre en el escritorio y llamare a la carpeta print, e.

 

2. Descripción.
 Este control permite digitar la descripción por cual se van a desactivar las ordenes.

3. Botón guardar.
El botón se habilitara cuando se seleccione un motivo y se digite una descripción, si da click en el botón se generaría una imagen similar a la siguiente:                                                      

[Imagen de referencia - Confirmación.]

Si acepta las dos advertencia mostrara un informe con la fecha de borrado, el usuario, el motivo, la descripción y las ordenes eliminadas,  una imagen similar a la siguiente:     

[Imagen de referencia - Listado de datos desactivados.]

                                                 

5. Botón cancelar. 
si se da click en el  botón se cancela la desactivación de las ordenes.

 

Resultado    

Es una herramienta que provee la aplicación para borrar el resultado y el comentario de las pruebas seleccionadas en el rango de ordenes sin importar su estado.                                           


[Imagen de referencia - Resultado.]

1. Fecha de  la orden.  
Usa el control para seleccionar el año de la orden que desea hacer la consulta, su valor por defecto es el año en que se encuentra.

2. Orden incial.
 Usa el control para digitar la orden inicial  manualmente, se puede ingresar los digitos de la orden y el sistema debe completar la orden con el mes y el día.

3. Orden final.
 Usa el control para digitar la orden final  manualmente, se puede ingresar los digitos de la orden y el sistema debe completar la orden con el mes y el día.

4. Buscador.
Se escribe o se introduce un valor alfa-numérico para filtrar elementos de la lista de pruebas. El filtro se da de manera dinámica; a medida que se vaya digitando un valor, la lista va mostrando las filas coincidentes con las columnas visibles.

5. Total pruebas. 
Este control muestra la cantidad de pruebas visualizadas en la lista.

6. Todos. 
Este control selecciona todas las pruebas visualizadas en la lista. 

7. Ordenamiento. 
Usa el control de ordenamiento para ordenar de menor a mayor o viceversa según el campo que desees ordenar solo al hacer click sobre el.  

8. listado de pruebas.  
Es una lista de datos que contiene tres campos: código , abreviatura y nombre de las pruebas que se encuentran activas. Dicha lista puede ser ordenada y/o filtrada por cada campo.

9. Botón borrar resultado.  
Si se da click en el  botón  y  el rango contiene resultados para borrar, y se crearon motivos de tipo modificación de resultado, se generaría una imagen similar a la siguiente: 

[Imagen de referencia - Motivo del borrado.]

1. Motivo.
 Este control permite seleccionar el motivo por el cúal se va a eliminar el resultado.

2. Descripción.
 Este control permite digitar la descripción por cual  se va a eliminar el resultado.

3. Botón guardar.
El botón se habilitara cuando se seleccione un motivo y se digite una descripción, si da click en el botón se generaría una imagen similar a la siguiente:                                                      

[Imagen de referencia - Confirmación.]

Si acepta las dos advertencia mostrara un informe con la fecha de borrado, el usuario, el motivo, la descripción y las ordenes eliminadas con sus pruebas,  una imagen similar a la siguiente:     

[Imagen de referencia - Listado de datos borrados.]

                                                 

5. Botón cancelar. 
si se da click en el  botón se cancela la eliminación de los resultados.
