Editor de código de barras

Permite configurar, crear y modificar el diseño de las etiquetas. proporciona al usuario un gran conjunto de herramientas, componenetes y herramientas para desarrolar etiquetas, diseñarlo visualmente y obtener una vista previa de ellos.

A continuación, se muestra una ventana típica con sus partes: 


[Imagen de referencia - Editor de código de barras.]

1.Listado código de barras.  
visualización de los diseños de etiquetas creadas  para configurar por defecto.  A continuación, se muestra una ventana típica con sus partes:

1.1 .Número de versión de la etiqueta creada 

1.2  Botón para editar la etiqueta.

1.3  Botón para seleccionar la etiqueta como predeterminada.

1.4 vizualización de la etiqueta.

 

2. Listado de estiqueta adicional.
visualización de los diseños de etiquetas creadas para seleccionar como etiqueta adicional.  

3 Boton nuevo .
Botón crear una nueva etiqueta de la lista seleccionada, al dar click se muestra una ventana típica con sus partes:  

3.1  Se visualizan los elementos que se pueden agregar, la etiqueta se agrega a al área de trabajo al dar click sobre elemento.

3.2  Área de trabajo, se visualiza el diseño de la etiqueta.

3.3  Datos generales de la etiqueta el número de la versión y las dimensiones de la etiqueta.

3.4  Propiedades del elemento de los elementos agregados

3.5  Botón que guarda los cambios y agrega en la lista seleccionada  

 

4 Listado de etiquetas de paciente .
Botón crear una nueva etiqueta para pacientes, esta etiqueta es utilizada en Homebound, al dar click se muestra una ventana típica con sus partes.
