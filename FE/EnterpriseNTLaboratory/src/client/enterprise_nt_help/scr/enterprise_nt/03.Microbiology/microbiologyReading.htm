Lectura de microbiología

Formulario para la consulta de órdenes que tengan pruebas de microbiología como así mismo el ingreso de antibiogramas y la creación de tareas.

A continuación, se muestra una ventana típica con sus partes:

 

1. Filtro de ordenes:

  • Rango: Filtros para buscar rápidamente los cultivos y directos que se deseen reportar, se puede aplicar por un rango de ordenes o un rango de fechas.
  • Orden y/o muestra: Para este filtro se debe ingresar el número de orden seguido el número de la muestra ya sea de forma manual o por medio de un
    lector externo de códigos de barras, el sistema deberá filtrar únicamente las pruebas asociadas a la orden y a la respectiva muestra, si se digita la orden sin muestra
    el sistema deberá mostrar todas las pruebas de microbiología de la orden digitada.

2. Filtro de ordenes por prueba y por reportar:
  • Prueba: En esta opción se puede seleccionar una prueba de microbiología específica, solo se muestran aquellas pruebas cuya muestra este asociado a microbiología.
  • Reportar: En esta opción se puede elegir el tipo de prueba a filtrar, los cultivos me filtran las pruebas principales o cultivos, los directos me permite filtrar únicamente los exámenes que ingresaron como procedimientos del cultivo principal.

3. Widget informativo Pruebas sin procesar: (ordenes pendientes de resultado) en esta sección se mostrará la cantidad de órdenes que contienen pruebas que están pendientes de resultado según el filtro aplicado en los ítems 1 y 2, al dar clic sobre esta tarjeta el listado de ordenes solo mostrara las que tengan alguna prueba pendiente de resultado.

4. Widget informativo Pruebas sin Pre-validación  (ordenes pendientes de prevalidacion) en esta sección se mostrara la cantidad de órdenes que contienen pruebas que están pendientes de prevalidar según el filtro aplicado en los ítems 1 y 2 , al dar clic sobre esta tarjeta el listado de ordenes solo mostrara las que tengan alguna prueba pendiente prevalidacion.

5. Widget informativo Pruebas Sin validación  (ordenes pendientes de validación) en esta sección se mostrara la cantidad de órdenes que contienen pruebas que están pendientes de validar según el filtro aplicado en los ítems 1 y 2 , al dar clic sobre esta tarjeta el listado de ordenes solo mostrara las que tengan alguna prueba pendiente validación.

 

6.Listado de ordenes : Listado que puede ser organizado ascendente o descendentemente, cada número de orden es un link para cargar los pruebas que pertenecen a la orden en la grilla de resultados.

7. Botón de tareas: Botón para visualizar un listado de las tareas asignadas a los cultivos de las ordenes según el filtro seleccionado. una vez una tarea sea impresa, esta no se volverá a generarse a menos que se de clic en el botón de reiniciar tareas.

8. Botón de tareas por paciente: Botón para visualizar un listado de las tareas agrupadas por paciente asignadas a los cultivos de las ordenes según el filtro seleccionado. una vez una tarea sea impresa, esta no se volverá a generarse a menos que se de clic en el botón de reiniciar tareas.

9. Botón pendientes de verificación: Botón para visualizar un listado de las ordenes que ingresaron al laboratorio con muestras de microbiología, pero que aún no han sido verificadas, por lo tanto, no se les puede realizar el registro de resultados.

10. Botón reiniciar tareas: Este botón permite reiniciar los listados de tareas que han sido impresas para que puedan volver a ser generadas.

12. Comentario: Este botón permite adicionar o editar el comentario del resultado de la prueba escogida:

12.6   Botón para guardar el comentario realizado.

 

13. Plantilla de resultados: Este botón permite ingresar resultados a través de plantillas que fueron configuradas previamente en el módulo de Configuración.

  nota: una vez se haya realizado el registro de resultados en una plantilla y esta sea modificada, los datos no se deben ver afectados.

 [Imagen de referencia plantilla de resultados]  

 

 

13.1  Comentario: Se visualiza el Tipo de resultado o comentario realizado con anterioridad.

13.2  Selección: Se logra visualizar 3 tipos de resultado (Normal, Positivo y Negativo), se puede seleccionar cualquier de estas 3 opciones dependiendo de los resultados.

13.3  Resultado: Se observan los resultados seleccionados en el punto anterior.

13.4  Referencia: Se visualiza el resultado el cual está por referencia en el sistema.

13.5  Reiniciar: Botón para borrar los datos que se han registrado en el momento.

13.6  Guardar: Botón para guardar las plantillas de resultados creadas.

 

14. Adjuntos: Este boton permite visualizar los archivos(pdf) e imagenes(jpg, jpeg, png) adjuntos a la orden y/o la prueba de una orden, como asi mismo adjuntar y/o eliminar archivos e imagenes(toma de foto o subir archivos ya existentes) a una orden y/o la prueba de una orden.

 nota: una orden y/o prueba pueden contener maximo 6 archivos adjuntos.

  • Se debe seleccionar un examen para lograr adjuntar archivos a la prueba

[Imagen de referencia adjuntos]  

14.1  Tipo de prueba: Este contiene el número de la orden y el código de prueba.

14.2  Información del paciente: Contiene el nombre del paciente y la identificación.

14.3  Información del paciente: Contiene la Edad y sexo del paciente.

14.4  Archivos permitidos: Debemos tener en cuenta el tipo de archivo para asi poder adjuntar documentos, tipos de archivos (JPG, JPEG, PNG, PDF).

14.5  Tomar fotografía: Con esta opción podrá tomar una fotografía y adjuntarla de forma inmediata

14.6  Adjuntar archivos: En esta opción podrá adjuntar los archivos deseados, teniendo en cuenta el tipo

14.7  Eliminar archivos adjuntados: En esta opción logrará eliminar los archivos adjuntados.

 

15. Repeticiones: Este botón permite ver a detalle un listado de todas las repeticiones que se le han realizado a la prueba escogida.  

 

[Imagen de referencia repeticiones]  

15.1  Seleccione el motivo por el cual hace la repetición.

15.2  Cometario por el cual se hace la repetición.

15.3  Aplica la repetición.

 

16. Consulta de repeticiones : Este botón permite ver a detalle un listado de todas las repeticiones que se le han realizado a la prueba escogida.  

[Imagen de referencia consulta de repetiociones]  

 

17. Trazabilidad:  Este botón permite ver un listado de la trazabilidad de las tareas asignadas a los medios de cultivo y a los procedimientos

 

 

18. Antibiograma: Botón para realizar el reporte de antibiogramas a la prueba de una orden. para que este botón funcione correctamente deben haberse realizado las siguientes configuraciones:
  • Haber configurado microorganismos.
  • Haber configurado antibióticos.
  • Haber asociado microorganismos a un examen de microbiología de acuerdo al proceso de detección microbiana para identificar cual es el tratamiento que se le debe dar microorganismo (configuración de antibiogramas)
  • Se requiere asignar antibióticos a un antibiograma de manera que se pueda identificar la reacción con el microorganismo. (configuración de antibiogramas
  • Se requiere valores de referencia para un antibiótico de manera que la sensibilidad se calcule automáticamente para los casos donde se utilice la técnica de lectura manual.

 

 

[Imagen de referencia antibiograma]  

 

18.1  Submuestra: se visualizan y se seleccionara las sub muestras deseadas.

18.2  Método: se visualizará y se logrará seleccionar el método de recolección.

18.3  Sitio anatómico: en este se podrá observar y seleccionar el sitio anatómico deseado.

18.4  Lista de microorganismos: en este podrá seleccionar el microorganismo deseado y este genera una tabla de información en la parte derecha de la pantalla.

18.5  Nuevo: este botón permite agregar un nuevo microorganismo mediante un check

18.6  Información: Se logrará visualizar detalles como la revalidación, usuario y fechas

18.7  Comentario: se podrá editar el comentario, ingresado anteriormente.

18.8  Tabla: esta es generada seleccionando un microorganismo, se puede agregar información tener en cuenta que en el campo de interpretación solo se podrá colocar 3 opciones (1, 2,3 las cuales son predeterminadas del sistema 

18.9  Guardar: este botón logra guardar la información del antibiograma creado.

 

19. Medio de cultivo: Botón para realizar el reporte de tareas a un medio de cultivo especifico en un destino específico.

 

[Imagen de referencia Medios de cultivo]   

19.1  Medios de cultivo: los cuales son creados desde la configuración y lograremos escoger el deseado, tener en cuenta que este campo es obligatorio.

19.5  Guardar: Botón que guarda los medios de cultivos creados
