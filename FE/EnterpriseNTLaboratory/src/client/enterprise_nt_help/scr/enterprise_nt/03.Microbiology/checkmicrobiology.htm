Verificación de microbiología

La verificación de  microbiología es la herramienta que provee la aplicación para hacer el ingreso de las muestras de microobiologia, ya sea por medio de un lector o manualmente y marcala como verificada en el LIS, se puede imprimir código de barras de la muestra, se puede ver la entrevista y actualizar los datos  

A continuación, se muestra una ventana típica con sus partes: 

[Imagen de referencia - verificación microbiología.]

1. Orden - muestra. 
Usa el control para escanear el código de barras de una muestra o ingresar  manualmente, se puede ingresar los dígitos de la orden, el separador de la muestra y el sistema debe completar la orden con el número de digitos de la orden que se configura en la llave de cantidad de digitos de la orden, el mes y el año y el separador de la muestra que se configura en la llave separador de muestras.

2. datos de paciente.  
Es una lista de los datos del paciente que se encuentran relacionados a la orden.  

3. Muestra.
 Nombre de la muestra que se encuentra relaciona.

4. Prueba.
 la prueba asociada a la muestra.

5. Submuestra.
 Selecciona  una submuestras.

6. Método de recolección. 
se selecciona un método que indica la técnica que se empleara en el proceso.

7. Sitio anatómico.
 Selecciona el sitio anatómico que corresponde al lugar físico en donde se toma la muestra.

8. Botón añadir sitio anatómico.
Este botón muestra los campos para crear un sitio anatómico, se generaría una imagen similar a la siguiente: 

[Imagen de referencia - Creación sitio anatómico.]

 

  1. Abreviatura del sitio anatómico, el cual no se puede repetir. Es un campo alfa-numérico que acepta mayúsculas y minúsculas, lo máximo son 15 caracteres, Es un campo requerido .
  2. Nombre del sitio anatómico a definir el cual no se puede repetir. Es un campo alfa-numérico que acepta mayúsculas y minúsculas, lo máximo son 60 caracteres, Es un campo requerido.
  3. Este botón permite crear un sitio anatómico. 
  4. Este botón cancela la creación de un sitio anatómico

9. Diagnóstico permanente .
Este control muestra el último diagnóstico permanente que se han ingresado a la orden.

10. Trazabilidad del diagnóstico permanente.
 Este control muestra la trazabilidad de los diagnosticos ingresados.

11. Expandir del diagnóstico permanente. 
Este control muestra en toda la pantalla el control de diagnóstico permanente, se pueden filtrar por reportados y no reportados.

12. Comentario de la orden. 
Este control muestra el último comentario de la orden que se han ingresado a la orden. 

13. Trazabilidad del comentario de la orden.
Este control muestra la trazabilidad de los comentarios de la orden.

14. Expandir el comentario de la orden.
Este control muestra en toda la pantalla el control de comentarios de la orden, se pueden filtrar por reportados y no reportados.

15. Comentario de microbiología.
 Este control muestra los comentarios de microbiología.

16. botón de añadir comentarios de microbiología. 
Este botón muestra los campos para crear un comentario de microbíologia, se generaría una imagen similar a la siguiente: 

[Imagen de referencia - Comentario de microbiología.]

 

  1.  Este control permite digitar el comentario de microbiología con texto enriquecido.
  2.  Este control permite eliminar los item de la lista ingresados.
  3.  Este control permite añadir item a la lista digitando la tecla enter o en el botón agregar. 
  4.  Este botón muestra el control anterior para agregar item a la lista

 

17. Trazabilidad del comentario de microbíologia.
 Este control muestra la trazabilidad de los comentarios de microbiología, se generaría una imagen similar a la siguiente: 

[Imagen de referencia - trazabilidad de microbiología.]

18. Expandir el comentario de microbíologia .
 Este control muestra en toda la pantalla el control de comentarios de microbiología, se pueden filtrar por reportados y no reportados,  se generaría una imagen similar a la siguiente:  

[Imagen de referencia - Comentario de microbiología.]

19. Editar el comentario de microbíologia.
Este botón permite editar el comentario de microbíologia seleccionado.

20. Eliminar el comentario de microbíologia.
Este botón permite eliminar el comentario de microbíologia seleccionado.

21. bloqueo del comentario de microbíologia. 
Este botón se muestra cuando el comentario se ingreso por otro usuario y no se permite ni eliminar, ni modificar.  

22. Botón verificar. 
Este boton verifica la muestra en microbiología.  

23. Botón de actualizar.  
Este boton actualiza  la submuestra, el método de recolección, el sitio anatómico y el comentario de microbiología.

24. Botón de código de barras.
 Este botón solo se habilitará cuando se haya ingresado una orden valida, al dar click en el se mostrara la siguiente información:

                        

[Imagen de referencia - Impresión de código de barras.]

 

  1. Listado de muestras que corresponden a la orden, las marcadas con el check son las que van a ser mandadas a la impresora de etiquetas.
  2. Cantidad de etiquetas que van a ser mandadas por muestra, el valor por defecto es el que se haya configurado para cada muestra.
  3. Check para imprimir una etiqueta adicional en el listado de muestras.
  4. Botón para mandar las etiquetas a la impresora de códigos de barras

Nota: Para que la impresión de etiquetas sea funcional y el sistema reconozca la impresora se debe tener instalado el cliente de impresión en la maquina donde se desee imprimir.

 Este botón visualiza la entrevista del ingreso para poder revisar si existen anomalias o se requieren hacer cambios en el proceso de la muestra, se generaría una imagen similar a la siguiente:
