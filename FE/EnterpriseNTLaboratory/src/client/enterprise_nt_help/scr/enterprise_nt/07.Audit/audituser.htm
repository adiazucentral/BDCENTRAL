Auditoría de usuarios

La auditoria de usuarios genera un reporte con las acciones realizadas por un usuario.

A continuación, se muestra una ventana típica con sus partes: 


[Imagen de referencia - Auditoria de usuarios.]

1. Fecha inicial. 
Usa el control seleccionar la fecha inicial con que consultara el informe.  

2. Fecha final.
Usa el control seleccionar la fecha final  con que consultara el informe.

3. Usuario.
Usa el control seleccionar el usuario con que consultara el informe.

4. imprimir auditoria.  
Se imprime una lista de valores modificados por el  usuario seleccionado.
