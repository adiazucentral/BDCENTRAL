Registro de resultados por lote
Este modulo nos permite el ingreso de resultados de una prueba por  rangos de ordenes.                    
A continuación, se muestra una ventana típica  con sus partes:

 

  1. Filtro por numero de orden, selecciona el rango menor y el rango mayor de las ordenes por las cuales desea filtrar. se llena automáticamente con la fecha del día si solo se ingresan los dígitos de la orden.
  2. Tipo de resultado de la prueba escogida, existen tres tipos:
      Resultado tipo texto
      Resultado tipo numerico
      Resultado tipo lista
  3. Lista de selección para la prueba a la que se le asignara el resultado (código, nombre).
  4. Campo para digitar o seleccionar el resultado, el comportamiento de este campo depende de el tipo de resultado, para las pruebas tipo texto podrá ingresar cualquier carácter alfanumérico, para las pruebas numéricas solo podrá ingresar números y la cantidad de decimales dependerá de la configuración de la prueba, para las pruebas tipo lista se cargara una lista de selección con los resultado literales configurados para la prueba.
  5. Lista de autocompletado de los comentario codificados configurados en el sistema, la busqueda se realiza mediante el código del comentario codificado, cuado se seleccione se agregara la descripción del comentario y el campo se limpiara. 
  6. Campo para indicar si el comentario codificado que se seleccione se inserta al principio o al final del comentario general.
  7. Comentario general que se agregara al resultado de la prueba.
  8. Botón para guardar el resultado a la prueba seleccionada y que este asignada a las ordenes que pertenecen al rango seleccionado, al dar click en el bóton el sistema mostrara un mensaje para que el usuario confirme la acción que va a realizar, al momento de confirmar la acción el sistema informara cuantas ordenes fueron afectadas.
