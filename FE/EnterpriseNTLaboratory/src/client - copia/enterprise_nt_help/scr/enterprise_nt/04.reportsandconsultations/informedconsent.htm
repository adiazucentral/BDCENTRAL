Consentimieto Informado

Módulo de consulta de consentimiento informados que permita buscar por historia o por orden y poder consultar los exámenes a los que se les haya aplicado consentimiento informado.

A continuación, se muestra una ventana típica con sus partes: 


[Imagen de referencia - Consentimiento informado.]

1. Filtrar por .  
Usa el control para seleccionar por que desea filtrar.

  • Historia:  Permite buscar las ordenes de la historia digitada. se puede seleccionar el tipo de documento si se encuentra configurada. Se generaria una imagen similar a la siguiente:    

 

[Imagen de referencia - Filtro historia.]

  • Número de orden:  Permite buscar por la orden especifica. Se generaria una imagen similar a la siguiente:     

 

[Imagen de referencia - Filtro número de orden.]

2. Buscador.
Se escribe o se introduce un valor alfa-numérico para filtrar elementos de la lista de ordenes. El filtro se da de manera dinámica; a medida que se vaya digitando un valor, la lista va mostrando las filas coincidentes con las columnas visibles.

3. Listado de ordenes.
 listado de ordenes  que cumple con los filtro seleccionados.

4. Listado de pruebas.
 Es una lista de pruebas  que contiene consentimiento informado contiene cuatro campos: código , nombre, fecha y usuario. Dicha lista puede ser filtrada por cada campo.

5. Ver consentimiento informado.
Muestra un reporte del consentimiento informado del exámen.

6. Datos de paciente.
 Este control muestra los datos del paciente de la orden seleccionada.
