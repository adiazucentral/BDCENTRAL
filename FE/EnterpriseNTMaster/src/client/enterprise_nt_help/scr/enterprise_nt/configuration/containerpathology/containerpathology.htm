Contenedores

Un contenedor de patología hace referencia a los tipos de envases para análisis, recepción, conservación, transporte y almacenamiento de muestras.

A continuación, se muestra una ventana típica con sus partes:

[Imagen 01 - Formulario de contenedores]

 

1. Buscador.

Escribe una palabra o frase en el cuadro de búsqueda para buscar un elemento. Dado que la búsqueda se inicia en cuanto empiezas a escribir, si por ejemplo escribes "B", todos los contenedores de patología cuyo nombre empiece por la letra B se mostrarán en la lista de contenedores.

2. Ordenamiento.
Usa el control de ordenamiento para ordenar de menor a mayor o viceversa según el campo que desees ordenar solo al hacer click  sobre él.  

3. Cantidad.
Este control muestra la cantidad de contenedores visualizados en la lista.

4. Fecha - Usuario de última actualización.
Aquí se despliega la fecha y el usuario que modificaron el contenedor que ha sido seleccionado.

5. Botón ayuda.
Aquí despliega la explicación de cada uno de los controles

6. Nombre.
Nombre del contenedor. Acepta un máximo de 16 caracteres es alfanumérico sin caracteres especiales y es requerido.

7. Imprimir Casete.
Indica si el contenedor impirime casete o no. Cuando se crea un nuevo contenedor este toma el valor de inactivo. Este campo es booleano y requerido

8. Estado.
Indica si el contenedor se encuentra activo o inactivo. Cuando se crea un nuevo contenedor este toma el valor de activo. Este campo es booleano y requerido

9. Imagen.
Muestra la foto del contenedor seleccionado. No es un campo requerido, solo acepta imágenes en formato png y jpg.

Para este campo hay que tener en cuenta el tamaño de la imagen, puesto que no hay un límite, si se adjunta una imagen muy grande o muy pequeña el sistema le dará un tratamiento a  la imagen para dejarla de un tamaño de 100px de alto por 100px de ancho y esto puede afectar la visualización de la misma.

10. Botón nuevo.
Este botón habilita las cajas de texto (excepto el interruptor de estado) para crear un nuevo contenedor. Si dichas cajas de texto tienen datos, se limpian. Al pulsarlo, se habilita el botón Cancelar y se deshabilitan los botones Editar y Reporte . Este botón por defecto está habilitado.

11. Botón editar.
Este botón permite editar el contenedor selecciono de la lista de datos y es habilitado cuando se selecciona un contenedor de la lista. Se cargan los datos en las cajas de texto y al pulsarlo el botón Editar, se habilitan todos los controles del formulario. Se habilitan los botones Guardar y Cancelar, y se deshabilita el botón Reporte.

12. Botón guardar.
Este botón permite guardar un contenedor ya sea creado o modificado. Al pulsarlo se deshabilitan los botones Editar, Cancelar y el propio botón, y se habilitan los botón Nuevo y Reporte.

13. Botón cancelar.
Este botón cancela, limpia y deshabilita las cajas de texto incluido el interruptor del estado. Deshabilita los botones Editar y Guardar, y habilita el botón Reporte.

14. Botón  Reporte.
Este botón imprime los contenedores que se encuentran en la lista de datos. Se muestra un reporte con las mismas columnas y los mismos registros filtrados.

15. Listado de contenedores.
Es una lista de datos que contiene dos campos: Nombre y Estado. Dicha lista puede ser ordenada y/o filtrada por cada campo. La fila seleccionada siempre se mostrará resaltada en azul claro.
