Antibiograma

Formulario que define la configuración de antibióticos donde se pueden definir los valores de referencia, unidades y lineas de tratamiento; estos valores determinana la manera como se va a proceder respecto a uno o varios microorganismos presentes en las muestras de los pacientes, apartir de esta configuración  se determinan tambien si se aplicarab reglas de supresión en la impresión de resultados y se determina si se hara control de calidad mediante reglas de wesgard

A continuación, se muestra una ventana típica con sus partes:

 

           

      1. Cuadro de texto de búsqueda.

Se escribe o se introduce un valor alfa-numérico para filtrar elementos de la lista de antibiogramas. El filtro se da de manera dinámica; a medida que se vaya digitando un valor, la lista va mostrando las filas coincidentes con las columnas visibles.

 

      2. Ordenamiento.

Se usa el ordenamiento por columnas de menor a mayor o viceversa, la lista se organiza ya sea alfabéticamente o por valor (falso o verdadero). El ícono que aparece en cada columna cambia según el orden.
 

      3.  Cantidad.

Este control muestra la cantidad de antibiogramas  visualizados en la lista de datos. Su valor cambia a medida que se vayan filtrando valores en el cuadro de búsqueda o cuando se introduce un nuevo antibiograma.

 

4.    Antibiótico.

Listado de los antibióticos creados previamente, que se asociaran al antibiograma.

A continuación, se muestra una ventana típica con sus partes:

 

       4.1. Cuadro de texto de búsqueda antibióticos .

Se escribe o se introduce un valor alfa-numérico para filtrar elementos de la lista de antibióticos. El filtro se da de manera dinámica; a medida que se vaya digitando un valor, la lista va mostrando las filas coincidentes con las columnas visibles

4.2. Ordenamiento.

       Se usa el ordenamiento por columnas de menor a mayor o viceversa, la lista se organiza ya sea alfabéticamente o por valor (falso o verdadero). El ícono que aparece en cada columna cambia según el orden.
4.3. Antibiótico

       Nombre de los antibioticos que se encuentran activos .

      4.4.  Unidad .

  Indica la unidad de medida empleada para definir la interpretación del resultado. este campo es informativo y aparece en los informes de resultados acompañandoloal valor de referencia.   

  4.5.  Linea.

  Indica la prioridad en que se mostrara los resultados según su complejidad,esto se conoce como normas de supresión y se aplican en la impresión de reportes solo existen 3 lineas.

  4.6.  Asignado.

  Indica si el antibiotico se encuentra asociado al antibiograma.   

 

 

      5. Fecha de última actualización - Usuario.                                                                                    

Aquí se muestra la fecha y el usuario que modificó el antibiograma seleccionado. Por defecto no aparece visible, solo cuando se selecciona una fila de la lista de antibiogramas.


      6.  Botón ayuda .

Aquí se despliega el manual de cada página.

 

7.     Microorganismo 

Listado de los microorganismos y pruebas  que se relacionan en el antibiograma.

A continuación, se muestra una ventana típica con sus partes:

 

       7.1 microorganismo .

Lista de los microorganismos  agregados para que se agrege el antibiograma 

7.2 prueba.        

Lista de las pruebas relacionados con el microorganismos para relacionar el antibiograma 

       7.3  Agregar  relación microorganismo-prueba.

       Botón que agrega una nueva relación para poder seleccionar una nueva relación microorganismo-prueba. 

7.4 Eliminar relación de microorganismo-prueba.

       Botón que elimina la relación seleccionada.

 

 

 

      8. Código.                                                                                                      

Es un campo alfa-numérico que hace referencia a la identificación única del antibiogramas, es un campo requerido, unico y  lo máximo son 10 caracteres

 

      9. Abreviatura.                                                                                                    

      Este campo indica la abreviatura que identifica el antibiograma, el cual se emplea en la asociación con el microorganismo, es un campo requerido, unico y  lo máximo son 60 caracteres.

      

8.   10. Antibiograma.                                                                                                      

Este campo indica el nombre del antibiograma, el cual se emplea en la impresión de resultados, es un campo requerido, unico y  lo máximo son 60 caracteres.

 

      11. Aplicar reglas de supresión.

      Este campo indica si se hará control de calidad mediante reglas de wesgard.

 

      12.  Estado.

Es un campo que permite activar o desactivar una tarifa . Al cambiar el estado muestra un cuadro de confirmación donde pregunta si se encuentra seguro de

cambiar el estado o no. 

                                                                                                                             

13.    Botón nuevo.

Este botón habilita las cajas de texto (excepto el interruptor de estado) para crear un nuevo antibiograma. Si dichas cajas de texto tienen datos, se limpian.

Al pulsarlo, se habilita el botón Cancelar y se deshabilitan los botones Editar e Imprimir. Este botón por defecto debe estar habilitado.

 

14.  Botón editar.

Este botón permite editar el antibiograma seleccionado de la lista de datos y es habilitado cuando se selecciona de la lista. Se cargan los datos en las cajas de texto y al pulsar el botón Editar, se habilitan todos los controles del formulario. Se habilitan los botones Guardar y Cancelar, y se deshabilita el botón Imprimir.

       

        15 .    Botón guardar.

Este botón permite guardar un antibiograma ya sea creado o modificado. Al pulsarlo se deshabilitan los botones Editar, Cancelar y el propio botón, y se habilitan los botones Nuevo e Imprimir. 

 

       16 .    Botón cancelar.

Este botón cancela, devuelve y deshabilita las cajas de texto incluido el interruptor del estado. Deshabilita los botones Editar y Guardar, y habilita el botones Imprimir y Nuevo.

 

       17 .    Botón reporte

Este botón abre una ventana donde se muestra el reporte de los antibiogramas que se encuentran en la lista de datos. Se muestra los mismos registros filtrados y las opciones para guardar ya sea en PDF o en Excel.

 

       18.    Listado de antibiogramas.

       Es una lista de datos que contiene 4 campos: código, abreviatura, antibiograma y estado. Dicha lista puede ser ordenada y/o filtrada por cada campo. La fila seleccionada siempre se mostrará resaltada en azul claro. Las filas inactivas se mostraran resaltadas en rosado.
