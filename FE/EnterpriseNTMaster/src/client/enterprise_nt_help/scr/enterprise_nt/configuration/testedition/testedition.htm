Edición de pruebas

Este formulario permite modificar algunos datos de las pruebas en forma rápiada; como son el área, el código, el nombre, unidad, y muestra entre otros.

A continuación, se muestra la ventana especial para la modificación de las pruebas por alguno de sus campos:

 

Al ingresar a dicho formulario, aparece un mensaje de advertencia informando que los cambios realizados pueden afectar la configuración del sistema donde existan pruebas.

1. Cuadros de texto de búsqueda.
Se escribe o se introduce un valor alfa-numérico para filtrar elementos de la lista de áreas ubicada al lado izquierdo y se realiza la misma operación para el caso de la lista de pruebas ubicada a la derecha. El filtro se da de manera dinámica; a medida que se vayan digitando caracteres, la lista va mostrando las filas coincidentes con las columnas visibles. 

2. Ordenamiento.

Se usa el ordenamiento por columnas de menor a mayor o viceversa, ambas listas se organizan ya sea alfabéticamente o por valor (falso o verdadero). El ícono que aparece en cada columna cambia según el orden. 

3. Cantidad.

Estos controles muestran la cantidad de registros de cada lista respectivamente. Su valor cambia a medida que se vayan filtrando valores en el cuadro de búsqueda. 

4. Fecha de última actualización - Usuario.                                                                                                        

Aquí se muestra la fecha y el usuario que modificó un examen o prueba correspondiente al área seleccionada. Por defecto no aparece visible, solo cuando se selecciona una fila de la lista de áreas.  

5. Botón ayuda.   

Aquí se despliega el manual de cada página.

 

6. Listado de áreas.
Es la lista de las áreas activos que se visualizan con los campos de Código y Nombre de área. Dicha lista puede ser ordenada y/o filtrada por cada campo. La fila seleccionada siempre se mostrará resaltada en azul claro y en ese momento aparecerán las pruebas asociadas al área seleccionada, dichas pruebas se visualizarán en una lista con los campos disponibles para editar.

7. Listado de pruebas a editar.
Al dar click en un registro de la lista de pruebas, éste habilita las columnas de Código, Abreviatura, Nombre, Área, Muestra y Unidad. La lista se puede filtrar y ordenar por cada columna. 

8. Editar código.
Se digita un código y guarda el cambio al perder el foco del control, se valida si el código está repetido o si se deja vacío.

9. Editar abreviatura.
Se digita una abreviatura y guarda el cambio al perder el foco del control, se valida si dicha abreviatura está repetida, si está vacía o si se digita una palabra reservada (GENDER, SIZE, WEIGHT, RACE) para las fórmulas de las pruebas.

10. Editar nombre.
Se digita un nombre de la prueba y guarda el cambio al perder el foco del control, se valida si el nombre está repetido o si se deja vacío.

11. Editar área.
Se selecciona el área a cambiar, se guarda el cambio al perder el foco del control; este cambio se ve reflejado cuando en la lista de áreas se selecciona el área donde se pasó la prueba, este cambio afecta el orden de impresión de las pruebas. Este campo siempre estará seleccionado con un valor.

12. Editar muestra.
Se selecciona la muestra a cambiar y guarda el cambio al perder el foco del control. Este campo siempre estará seleccionado con un valor.

13.  Editar unidad.
Se selecciona la unidad cambiar y guarda el cambio al perder el foco del control. Este campo puede estar vacío por lo que no siempre aparecerá seleccionado.
