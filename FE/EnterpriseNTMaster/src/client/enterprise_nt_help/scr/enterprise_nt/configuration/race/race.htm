Razas

Este maestro define las diferentes razas, las cuales sirven para definir valores de referencia en los pacientes y por grupos étnicos. Se emplea en el ingreso de órdenes.
A continuación, se muestra una ventana típica con sus partes:

 


 
1. Ordenamiento.
Se usa el ordenamiento por columnas de menor a mayor o viceversa, la lista se organiza ya sea alfabéticamente, numéricamente. El ícono que aparece en cada columna cambia según el orden.

2. Cuadro de búsqueda.                                                                                           
Se escribe o se introduce un valor alfa-numérico para filtrar elementos de la lista de razas. El filtro se da de manera dinámica; a medida que se vaya digitando un valor, la lista va mostrando las filas coincidentes con las columnas visibles.

3. Cantidad.
Este control muestra la cantidad de razas visualizados en la lista de datos. Su valor cambia a medida que se vayan filtrando valores en el cuadro de búsqueda o cuando se introduce una nueva raza. 

4. Usuario- fecha de última actualización.                                                                                                      
Aquí se muestra la fecha y el usuario que modificaron la raza que ha sido  seleccionado. Por defecto no aparece visible, solo cuando se selecciona una  fila de la lista de razas 

5. Botón ayuda.
Aquí se despliega el manual de cada página. 

6. Código. 
Letra que distingue el llamado o selección de la raza, el cual no se puede repetir, acepta mayúsculas y minúsculas, lo máximo es un carácter es un dato que siempre es requerido . 

7. Representación numérica.
valor que se utilizará en las fórmulas para el cálculo del resultado, es un campo numerico no requerido y su valor por defecto es 1.

8. Raza.
Nombre de la raza a definir. el cual no se puede repetir, es un campo alfa-numérico que acepta mayúsculas y minúsculas, lo máximo son 60 caracteres, Es un campo requerido.

9. Estado.
Este botón Indica si registro está activo o no, al modificar el estado siempre hay aparece un cuadro de dialogo que pide la confirmación.

10. Botón nuevo.
Este botón habilita las cajas de texto (excepto el interruptor de estado) para crear un nueva raza. Si dichas cajas de texto tienen datos, se limpian. Al pulsarlo, se habilita el botón Cancelar y se deshabilitan los botones Editar e Imprimir. Este botón por defecto debe estar habilitado.

11. Botón editar.
Este botón permite editar la raza seleccionada de la lista de datos y es habilitado cuando se selecciona una raza de la lista. Se cargan los datos en las cajas de texto y al pulsarlo el botón Editar, se habilitan todos los controles del formulario. Se habilitan los botones Guardar y Cancelar, y se deshabilita el botón Imprimir .

12. Botón guardar.
Este botón permite guardar un raza ya sea creada o modificada. Al pulsarlo se deshabilitan los botones Editar, Cancelar y el propio botón, y se habilitan los botones Nuevo e Imprimir . 

13. Botón cancelar.
Este botón cancela, limpia y deshabilita las cajas de texto incluido el interruptor del estado. Deshabilita los botones Editar y Guardar, y habilita el botón Imprimir.

14. Botón reporte.
Este botón abre una ventana emergente que muestra el informa de Razas, perimite exportar en PDF, EXCEl o imprimir las razas que se encuentran en la lista de datos con las mismas columnas y los mismos registros filtrados.

15. Lista de razas.
Es una lista de datos que contiene dos campos: Raza y Estado. Dicha lista puede ser ordenada y/o filtrada por cada campo. La fila seleccionada siempre se mostrará resaltada en azul claro y las razas inactivas estarán resaltadas con rosado.
