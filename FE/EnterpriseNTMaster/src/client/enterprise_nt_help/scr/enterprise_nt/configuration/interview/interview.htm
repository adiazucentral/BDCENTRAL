Entrevista

El maestro de entrevista agrupa una serie de preguntas para asociársela a cada uno de las pruebas, o general a la orden, al laboratorio o para control de pánicos  

A continuación, se muestra una ventana típica con sus partes:

1.     Cuadro de búsqueda.                                                                                          

Se escribe o se introduce un valor alfa-numérico para filtrar elementos de la lista de datos. El filtro se da de manera dinámica; a medida que se vaya digitando un valor, la lista va mostrando las filas coincidentes con las columnas visibles.

 

2.     Ordenamiento.

Se usa el ordenamiento por columnas de menor a mayor o viceversa, la lista se organiza ya sea alfabéticamente, numéricamente. El ícono que aparece en cada columna cambia según el orden.

 

3.     Cantidad.  

Este control muestra la cantidad de entrevistas visualizadas en la lista de datos. Su valor cambia a medida que se vayan filtrando valores en el cuadro de búsqueda o cuando se introduce un nueva entrevista. 

 

    4.     Usuario- fecha de última actualización.                                                                                                      

     Aquí se muestra la fecha y el usuario que modificaron la entrevista que ha sido   seleccionada. Por defecto no aparece visible, solo cuando se selecciona una  fila de la lista.

 

     5.      Botón ayuda.

Aquí se despliega el manual de cada página.

 

6.    Tipo.

Indica si la entrevista aplica a la orden, a pruebas, remisiones o destinod. Es un campo requerido.

 

7.     Pánico.

Indica si la entrevista creada aplica para pruebas con resultado en pánico aplica para todos los tipos de entrevista solo puede existir una entrevista con esta campo habilitado .

 

8.     Entrevista .

Nombre de la entrevista, el cual no se puede repetir. Es un campo alfa-numérico que acepta mayúsculas y minúsculas, lo máximo son 60 caracteres, Es un campo requerido.

 

9.     Consentimiento informado.

Indica si la entrevista creada aplica para consentimiento informado .

 

10.  Estado.

Este botón Indica si registro está activo o no, al modificar el estado siempre hay aparece un cuadro de dialogo que pide la confirmación.

12.  Preguntas.

      Panel  para indicar las preguntas que van asociadas a la entrevista.    A continuación, se muestra una ventana típica con sus partes:

 

          12 .1 Crear pregunta. 

   Boton para crear una pregunta no existente en el listado, al dar click en el boton se visualizara una ventana como la siguiente:

12.1.1 Boton para guardar la pregunta.

12.1.2 Boton para cancelar la accion y volver al listado de preguntas

12.1.3 Codigo de la pregunta a crear

12.1.4 Nombre de la pregunta a crear

12.1.5 Control que va a ser asignado a la pregunta, al escoger las opciones de lista de seleccion o lista de seleccion multiple la vista cambiara y se mostraran las opciones de respuesta para la pregunta.

en esta vista podra crear nuevas opciones de respuesta o escoger una que haya sido creada con anterioridad.

12.2 Busqueda de preguntas. 

Opcion para la busqueda de preguntas asociadas y no asociadas en una entrevista

12.3 Ordenamiento. 

Ordenamiento de las preguntas asociadas a una entrevista este podra ser modificado seleccionanado la pregunta y moviendola hasta la posicion deseada

12.4 Control de la pregunta. 

Campo informativo para el tipo de control que tiene asociado la pregunta.

12.5 Preguntas no asociadas. 

Listado de las preguntas no asociadas a la entrevista nueva o editada.

12.6 Preguntas asociadas. 

Listado de las preguntas asociadas a una entrevista nueva o editada.

 

 

     13.  Botón nuevo.

Este botón habilita las cajas de texto (excepto el interruptor de estado) para crear una nueva entrevista. Si dichas cajas de texto tienen datos, se limpian.Al pulsarlo, se habilita el botón Cancelar y se deshabilitan los botones Editar y Reporte. Este botón por defecto debe estar habilitado.

 

14.  Botón editar.

Este botón permite editar la entrevista seleccionada de la lista de datos y es habilitado cuando se selecciona una entrevista de la lista. Se cargan los datos en las cajas de texto y al pulsarlo el botón Editar, se habilitan todos los controles del formulario. Se habilitan los botones Guardar y Cancelar, y se deshabilita el botón Reporte.

 

15.  Botón guardar.

Este botón permite guardar una entrevista ya sea creada o modificada. Al pulsarlo se deshabilitan los botones Editar, Cancelar y el propio botón, y se habilitan los botones Nuevo y Reporte.

 

16.  Botón cancelar.

Este botón cancela, limpia y deshabilita las cajas de texto incluido el interruptor del estado. Deshabilita los botones Editar y Guardar, y habilita el botón Reporte.

 

17.  Botón Reporte .

Este botón imprime las entrevista seleccionada.

 

18.  Lista de entrevista.

Es una lista de datos que contiene tres campos: entrevista, tipo y Estado. Dicha lista puede ser ordenada y/o filtrada por cada campo. La fila seleccionada siempre se mostrará resaltada en azul claro y las inactivas estarán seleccionadas con rosado.
