Ordenamiento demográficos

El maestro de ordenamiento demográficos se usa para .
A continuación, se muestra una ventana típica con sus partes:


1. Origen del demográfico.                                                                                            
Es un campo de selección única, donde aparecen dos opciones para escoger:
 * Historia (H) es para que el demográfico aparezca en la historia de un paciente.
 * Orden (O) es para que el demográfico aparezca en el ingreso de órdenes.

2. listado demográficos.
Es una lista de datos que contiene 2 campos:  N° ordenamiento, Nombre,. dicha lista puede cambiar el orden arrastrando los ítems como se quieren en qué orden quiere que  muestren los datos.
