Excluir pruebas por demográfico

El maestro de excluir pruebas por demográfico permite seleccionar las pruebas que no se van a procesar por cada uno de los demográficos.  

A continuación, se muestra una ventana típica con sus partes:

 

1. Ordenamiento.

Se usa el ordenamiento por columnas de menor a mayor o viceversa, la lista se organiza ya sea alfabéticamente, numéricamente. El ícono que aparece en cada columna cambia según el orden.  

      2. Cuadro de búsqueda de los ítems  demográfico.                                                                                         

Se escribe o se introduce un valor alfa-numérico para filtrar elementos de la lista de ítems demográficos. El filtro se da de manera dinámica; a medida que se vaya digitando un valor, la lista va mostrando las filas coincidentes con las columnas visibles.

 

3. Demográfico.

Indica el demográfico al cual se aplicara el filtro, este demográfico es configurable en configuración. 

4. Cantidad.

Este control muestra la cantidad de datos visualizados en la lista de datos. Su valor cambia a medida que se vayan filtrando valores en el cuadro de búsqueda.

 

5. Cuadro de búsqueda de pruebas.                                                                                         

Se escribe o se introduce un valor alfa-numérico para filtrar elementos de la lista de pruebas. El filtro se da de manera dinámica; a medida que se vaya digitando un valor, la lista va mostrando las filas coincidentes con las columnas visibles. 

 

6. Botón ayuda.

Aquí se despliega el manual de cada página.

 

7. Lista de pruebas.

Lista de datos que contiene tres campos: Código, nombre y excluir. Dicha lista puede ser ordenada y/o filtrada por cada campo. La fila seleccionada siempre se mostrará re saltada en gris claro y esta lista es para excluir las pruebas del demográfico seleccionado.

 

8.  Botón guardar.

Este botón permite guardar la relación entre prueba y el ítem demográfico seleccionado. Al pulsarlo se deshabilita el botón Cancelar y el propio botón, y se habilita el  botón  reporte.

 

9.  Botón cancelar.

Este botón cancela y vuelve los datos como se encontraban guardados. Deshabilita el botón Guardar y habilita el botón Reporte

 

10.  Botón reporte.

Este botón imprime las pruebas asociadas al ítem demográfico que se encuentra seleccionado

 

11. Lista de ítem demográfico.

Es una lista de datos que contiene los ítems demográficos asociados al demográfico que está configurado. Dicha lista puede ser ordenada y/o filtrada. La fila seleccionada siempre se mostrará re saltada en azul.
