Laboratorio

Formulario que define los laboratorios en donde se realizarán los exámenes ya sean manualmente o por analizador.

A continuación, se muestra una ventana típica con sus partes:

 

1.     Cuadro de búsqueda.                                                                                           

Se escribe o se introduce un valor alfa-numérico para filtrar elementos de la lista de datos de laboratorios. El filtro se da de manera dinámica; a medida que se vaya digitando un valor, la lista va mostrando las filas coincidentes con las columnas visibles.

 

2.     Ordenamiento.

Se usa el ordenamiento por columnas de menor a mayor o viceversa, la lista se organiza ya sea alfabéticamente, numéricamente. El ícono que aparece en cada columna cambia según el orden.

 

3.     Cantidad.

Este control muestra la cantidad de laboratorios visualizados en la lista de datos. Su valor cambia a medida que se vayan filtrando valores en el cuadro de búsqueda o cuando se introduce un nuevo laboratorio.

 

 

4.     Usuario- fecha de última actualización.                                                                                                     

Aquí se muestra la fecha y el usuario que modificaron el laboratorio que ha sido  seleccionada. Por defecto no aparece visible, solo cuando se selecciona una  fila de la lista de laboratorios

 

                  

5.     Botón ayuda.

Aquí se despliega el manual de cada página.

 

 

6.     Código.

Identificación del laboratorio el cual no se puede repetir. Es un campo numérico; lo máximo son 5 caracteres y es un dato que siempre será requerido.

 

 

7.     Laboratorio.  

Nombre del laboratorio, el cual debe ser único.Es un campo alfa-numérico que acepta mayúsculas y minúsculas, lo máximo son 60 caracteres, Es un campo requerido.

 

 

8.     Dirección.

Este campo indica la dirección de contacto del laboratorio. Es un campo alfa-numérico que acepta mayúsculas y minúsculas, lo máximo son 60 caracteres.

 

 

9.     Teléfono.

Este campo indica el número telefónico para contactar el laboratorio. Es un campo configurable en configuración.

 

 

10.  Tipo

Este campo Indica si el laboratorio pertenece a la red del laboratorio de la entidad o es un laboratorio al cual se remiten las muestras, indicando en cada caso si es interno o externo.

 

 

11.  Contacto.

Este campo indica el nombre del  responsable del laboratorio. Es un campo alfa-numérico que acepta mayúsculas y minúsculas, lo máximo son 60 caracteres.

 

 

12.  Ruta de exportación.

Para los casos donde se envía información por medio de archivos planos a otros sistemas de laboratorios, esta ruta (física del equipo) indica en donde se  almacenarían para que puedan ser utilizados.

 

13.  Estado.

Este botón Indica si registro está activo o no, al modificar el estado siempre  aparece un cuadro de dialogo que pide la confirmación.

 

 

14.  Botón nuevo.

Este botón habilita las cajas de texto (excepto el interruptor de estado) para crear una nuevo laboratorio. Si dichas cajas de texto tienen datos, se limpian.

Al pulsarlo, se habilita el botón Cancelar y se deshabilitan los botones Editar y Reporte. Este botón por defecto debe estar habilitado.

 

 

15.  Botón editar.

Este botón permite editar el laboratorio seleccionado de la lista de datos y es habilitado cuando se selecciona un laboratorio de la lista. Se cargan los datos en las cajas de texto y al pulsarlo el botón Editar, se habilitan todos los controles del formulario. Se habilitan los botones Guardar y Cancelar, y se deshabilita el botón Reporte.

 

 

16.  Botón guardar.

Este botón permite guardar un laboratorio ya sea creada o modificada. Al pulsarlo se deshabilitan los botones Editar, Cancelar y el propio botón, y se habilitan los botones Nuevo y Reporte.

 

 

17.  Botón cancelar.

Este botón cancela, limpia y deshabilita las cajas de texto incluido el interruptor del estado. Deshabilita los botones Editar y Guardar, y habilita el botón Reporte.

 

 

18.  Botón Reporte .

Este botón imprime los laboratorios que se encuentran en la lista de datos. Se muestra un reporte con las mismas columnas y los mismos registros filtrados.

 

 

19.  Lista de laboratorio.

Es una lista de datos que contiene cuatro campos: Código, laboratorio, tipo  y Estado. Dicha lista puede ser ordenada y/o filtrada por cada campo. La fila seleccionada siempre se mostrará resaltada en azul claro y las muestras inactivas estarán seleccionadas con rosado.
