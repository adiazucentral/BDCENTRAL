Días alarma pendiente de pago

El maestro de días para alarma define la cantidad de días que se buscará una prueba por demográfico hacia atrás, para verificar si existen pagos pendientes en la facturación de Estados Unidos  

A continuación, se muestra una ventana típica con sus partes:

 

 

1. Ordenamiento.

Se usa el ordenamiento por columnas de menor a mayor o viceversa, la lista se organiza ya sea alfabéticamente, numéricamente. El ícono que aparece en cada columna cambia según el orden.

 

2. Cuadro de búsqueda ítem demográfico.                                                                                          

Se escribe o se introduce un valor alfa-numérico para filtrar elementos de la lista de ítem demográfico.  El filtro se da de manera dinámica; a medida que se vaya digitando un valor, la lista va mostrando las filas coincidentes con las columnas visibles. 

 

3. Cantidad. 

Este control muestra la cantidad de datos visualizados en la lista. Su valor cambia a medida que se vayan filtrando valores en el cuadro de búsqueda .

 

4. Usuario- fecha de última actualización.                                                                                                                                                                                     

Aquí se muestra la fecha y el usuario que modificaron el usuario   que ha sido  seleccionado. Por defecto no aparece visible, solo cuando se selecciona una  fila de la lista

 

5. Botón ayuda.

Aquí se despliega el manual de cada página.

 

6. Área.

Elije el área  por desea filtrar la prueba, cuando selecciona una área se cargan las pruebas asociados a esa área.

 

7. Cuadro de búsqueda pruebas.                                                                                         

Se escribe o se introduce un valor alfa-numérico para filtrar elementos de la lista de pruebas.  El filtro se da de manera dinámica; a medida que se vaya digitando un valor, la lista va mostrando las filas coincidentes con las columnas visibles.

 

8. Lista de pruebas.

Es una   lista de datos que contiene tres campos: Código, nombre y cantidad de días. Dicha lista puede ser ordenada y/o filtrada por cada campo. La fila seleccionada siempre se mostrará re saltada en azul claro.

 

9. Botón guardar.

Este botón permite guardar los días de alarma.

 

   10. Botón cancelar.

Este botón nos permite devolvernos a un estado anterior de la lista mientras no se    haya guardado.

 

11. Botón reporte.

Este botón imprime los días de alarma asociadas al demográfico que se encuentran en la lista de datos.  

 

12. Lista de ítem demográfico.

Es una   lista de datos que contiene el nombre del demográfico. Dicha lista puede ser ordenada y/o filtrada por cada campo. La fila seleccionada siempre se mostrará re saltada en azul claro.
