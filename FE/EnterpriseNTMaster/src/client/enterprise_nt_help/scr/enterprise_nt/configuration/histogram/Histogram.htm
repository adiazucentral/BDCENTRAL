Histograma

Configura las variables que el sistema téndrá en cuenta para generar el histograma grafico de estadisticas.

 

1. Cuadro de búsqueda.                                                                                           

Se escribe o se introduce un valor alfa-numérico para filtrar elementos de la lista de histogramas. El filtro se da de manera dinámica; a medida que se vaya digitando un valor, la lista va mostrando las filas coincidentes con las columnas visibles.

 

2. Ordenamiento. 

Se usa el ordenamiento por columnas de menor a mayor o viceversa, la lista se organiza ya sea alfabéticamente, numéricamente. El ícono que aparece en cada columna cambia según el orden.

 

3. Cantidad. 

Este control muestra la cantidad de histogramas visualizados en la lista de datos. Su valor cambia a medida que se vayan filtrando valores en el cuadro de búsqueda o cuando se introduce un nuevo histograma.

                                                                                                          

4. Usuario- fecha de última actualización.                       

Aquí se muestra la fecha y el usuario que modificaron el histograma que ha sido  seleccionado. Por defecto no aparece visible, solo cuando se selecciona una  fila de la lista de histogramas.

 

5.  Botón ayuda. 

Aquí se despliega el manual de cada página. 

 

6. Nombre.

Nombre que identifica el histograma,  no se puede repetir. Es un campo alfa-numérico que acepta mayúsculas y minúsculas, lo máximo son 64 caracteres, Es un campo requerido.

 

7.  Rango mínimo.

Rango mínimo que aplica para el histograma.

 

8.  Rango máximo.

Rango maximo que aplica para el histograma.

 

9.  Estado.

Es un campo en forma de interruptor que permite activar o desactivar un histograma. Al cambiar el estado muestra un cuadro de confirmación donde   pregunta si se encuentra seguro de cambiar el estado o no.

 

10. Botón nuevo. 

Este botón habilita las cajas de texto (excepto el interruptor de estado) para crear un nuevo histograma. Si dichas cajas de texto tienen datos, se limpian. Al pulsarlo, se habilita el botón Cancelar se deshabilitan los botones Editare Reporte. Este botón por defecto debe estar habilitado.

 

11. Botón editar.

Este botón permite editar el histograma seleccionado de la lista de datos y es habilitado cuando se selecciona de la lista. Se cargan los datos en las cajas de texto y al pulsar el botón Editar, se habilitan todos los controles del formulario. Se habilitan los botones Guardar y Cancelar, y se deshabilita el botón Reporte.

 

12. Botón guardar.

Este botón permite guardar un histograma ya sea creado o modificado. Al pulsarlo se deshabilitan los botones Editar, Cancelar y el propio botón, y se habilitan los botones Nuevo e Reporte.

 

13.  Botón cancelar.

Este botón cancela, devuelve y deshabilita las cajas de texto incluido el interruptor del estado. Deshabilita los botones Editar y Guardar, y habilita el botón Reporte y Nuevo.

 

14.  Botón Reporte

Este botón abre una venta el reporte de histogramas que se encuentran en la lista de datos. Se muestra con los mismos registros filtrados y con la opción de exportar a Excel, PDF u otros formatos.

 

15.  Lista de histogramas

Es una lista de datos que contiene cuatro campos: Código, rango mínimo, rango máximo y estado. Dicha lista puede ser ordenada y/o filtrada por cada campo. La fila seleccionada siempre se mostrará resaltada en azul claro y  las inactivas estarán resaltadas con rosado.
