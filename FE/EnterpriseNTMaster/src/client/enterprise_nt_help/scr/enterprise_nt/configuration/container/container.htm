Recipientes

Un recipiente hace referencia a la distribución interna del laboratorio y segmenta la especialidad relacionada con el procesamiento de la muestra, ejemplos de áreas son química, microbiología, hormonas, citología, patología, etc.

A continuación, se muestra una ventana típica con sus partes:

1. Listado de recipientes.
Es una lista de datos que contiene tres campos: Prioridad, Nombre  y Estado. Dicha lista puede ser ordenada por prioridad dando click sostenido sobre el elemento y moverlo hacia la posicion que desee. La fila seleccionada siempre se mostrará resaltada en azul claro. El registro inactivo se mostrará en rosado.

2. fecha ultima actualizacion.

Fecha y usuario que inserto o actualizo el registro seleccionado

3. Botón ayuda.  
Despliegue del manual de cada página. 

4. Nombre.
Identificador único del recipiente. Acepta un máximo de 60 caracteres, este campo es requerido y alfanumérico.

5. Imagen.
Muestra la foto del recipiente seleccionado. No es un campo requerido, solo acepta imágenes en formato png y jpg

Para este campo hay que tener en cuenta el tamaño de la imagen, puesto que no hay un límite, si se adjunta una imagen muy grande o muy pequeña el sistema le dará un tratamiento a  la imagen para dejarla de un tamaño de 100px de alto por 100px de ancho y esto puede afectar la visualización de la misma.

6. Unidad.

Unidad de referencia para las mediciones del tubo.

7. Estado. 
Indica si el recipiente se encuentra activa o inactiva. Cuando se crea una nueva área este toma el valor de activo. Este campo es en forma de interruptor.

8. Botón Reporte .
Este botón imprime los recipientes que se encuentran en la lista de datos. Se muestra un reporte con las mismas columnas y los mismos registros filtrados.

9. Botón cancelar.
Este botón cancela, limpia y deshabilita las cajas de texto incluido el interruptor del estado. Deshabilita los botones Editar y Guardar, y habilita el botón Reporte.

10. Botón guardar.
Este botón permite guardar un recipiente ya sea creada o modificada. Al pulsarlo se deshabilitan los botones Editar, Cancelar y el propio botón, y se habilitan los botón Nuevo y Reporte.

11. Botón editar.
Este botón permite editar la área selecciona de la lista de datos y es habilitado cuando se selecciona un recipiente de la lista. Se cargan los datos en las cajas de texto y al pulsarlo el botón Editar, se habilitan todos los controles del formulario. Se habilitan los botones Guardar y Cancelar, y se deshabilita el botón Reporte.

12. Botón nuevo.
Este botón habilita las cajas de texto (excepto el interruptor de estado) para crear un nuevo recipiente. Si dichas cajas de texto tienen datos, se limpian. Al pulsarlo, se habilita el botón Cancelar y se deshabilitan los botones Editar y Reporte. Este botón por defecto está habilitado.
