Destinos de microbiología

Define los destinos en donde se realizan las tareas que se aplican a una muestra.

 

 

1.     Ordenamiento.

Se usa el ordenamiento por columnas de menor a mayor o viceversa, la lista se organiza ya sea alfabéticamente, numéricamente. El ícono que aparece en cada columna cambia según el orden.

 

2.    Cuadro de búsqueda.                                                                                           

Se escribe o se introduce un valor alfa-numérico para filtrar elementos de la lista de destinos de microbiología. El filtro se da de manera dinámica; a medida que se vaya digitando un valor, la lista va mostrando las filas coincidentes con las columnas visibles.

 

3.     Cantidad.

Este control muestra la cantidad de destinos de microbiología visualizados en la lista de datos. Su valor cambia a medida que se vayan filtrando valores en el cuadro de búsqueda o cuando se introduce un nuevo destino de microbiología.          

                                                                                                   

4.     Usuario- fecha de última actualización.                                                                                                     

Aquí se muestra la fecha y el usuario que modificaron el destino de microbiología  que ha sido  seleccionado. Por defecto no aparece visible, solo cuando se selecciona una  fila de la lista de destinos de microbiología. 

 

5.     Botón ayuda.

Aquí se despliega el manual de cada página. 

 

6.     Código.

Código que identifica el destino de microbiología,  el cual no se puede repetir. Es un campo alfa-numérico que acepta mayúsculas y minúsculas, lo máximo son 10 caracteres, Es un campo requerido.

 

7.     Destino.

Nombre del destino de microbiología,el cual no se puede repetir Es un campo alfa-numérico que acepta mayúsculas y minúsculas, lo máximo son 60 caracteres, Es un campo requerido.

 

8.   Reportar  tareas:

      Campo que indica si en el destino seleccionado se reportan las tareas de microbiología.

 

9.     Estado

Este botón Indica si registro está activo o no, al modificar el estado siempre hay aparece un cuadro de dialogo que pide la confirmación.

 

10. Usuarios-analizador

Lista de usuarios de tipo analizador para asignarlos al destino, esta lista se puede ordernar y ser filtrada por cada campo. 

 

11.  Botón nuevo

Este botón habilita las cajas de texto (excepto el interruptor de estado) para crear una nuevo destino. Si dichas cajas de texto tienen datos, se limpian. Al pulsarlo, se habilita el botón Cancelar y se deshabilitan los botones Editar e Reporte. Este botón por defecto debe estar habilitado.

 

12.  Botón editar

Este botón permite editar los destinos seleccionados de la lista de datos y es habilitado cuando se selecciona un destino  de la lista. Se cargan los datos en las cajas de texto y al pulsarlo el botón Editar, se habilitan todos los controles del formulario. Se habilitan los botones Guardar y Cancelar, y se deshabilita el botón Reportar.

 

13.  Botón guardar

Este botón permite guardar un destino  ya sea creado o modificada. Al pulsarlo se deshabilitan los botones Editar, Cancelar y el propio botón, y se habilitan los botones Nuevo e Reportar.

 

14.  Botón cancelar

Este botón cancela, limpia y deshabilita las cajas de texto incluido el interruptor del estado. Deshabilita los botones Editar y Guardar, y habilita el botón Reporte.

 

15.  Botón Reporte

Este botón imprime los destinos que se encuentran en la lista de datos. Se muestra un reporte con las mismas columnas y los mismos registros filtrados.

 

16.  Lista de destinos de microbiología

Es una lista de datos que contiene cuatro campos: Código, Destinó, reportar tareas y Estado. Dicha lista puede ser ordenada y/o filtrada por cada campo. La fila seleccionada siempre se mostrará resaltada en azul claro y  las inactivas estarán resaltadas con rosado.
