Pregunta

En este maestro se configuran las preguntas y respuestas utilizadas en las entrevistas del sistema

A continuación, se muestra una ventana típica con sus partes:

      1.     Cuadro de texto de búsqueda.

Se escribe o se introduce un valor alfa-numérico para filtrar elementos de la lista de datos de las preguntas. El filtro se da de manera dinámica; a medida que se vaya digitando un valor, la lista va mostrando las filas coincidentes con las columnas visibles.

 

      2.    Ordenamiento.

Se usa el ordenamiento por columnas de menor a mayor o viceversa, la lista se organiza ya sea alfabéticamente, numéricamente o por valor (falso o verdadero). El ícono que aparece en cada columna cambia según el orden.

 

      3.    Cantidad. 

Este control muestra la cantidad de registros visualizados en la lista de datos. Su valor cambia a medida que se vayan filtrando valores en el cuadro de búsqueda o cuando se introduce una nueva pregunta.

 

      4.    Fecha de última actualización - Usuario.

Aquí se muestra la fecha y el usuario que creó o modificaron el registro que ha sido seleccionado. Por defecto no aparece visible, solo cuando se selecciona una fila de la lista de las hojas de trabajo.

 

      5.    Botón ayuda. 

Aquí se despliega el manual de cada página.

 

6.   Codigo.

Nombre para identificar la pregunta. es un campo alfanumerico, con una longitud de 60 caracteres, unico y requerido

   

7.  Nombre.  

Contenido que se mostrará como pregunta en la entrevista (Lo que se consulta al entrevistado), es un campo alfanumerico, con una longitud de 255 caracteres, unico y requerido

 

8. Control.

Indica que tipo de control que se empleará dependiendo del tipo.

Preguntas abiertas:

  • Texto
  • Númerico
  • Comentario
  • Fecha

Preguntas cerradas: 

  • Combobox
  • Combo multiple

9.    Estado.

Indica si la pregunta se encuentra activa o no. Al cambiar el estado muestra un cuadro de confirmación donde pregunta si se encuentra seguro de cambiar el estado o no. El control tiene forma de interruptor (switch).

 

10.  Listado de respuestas .

Listado de las posibles respuestas que puede tener una pregunta, este solo se mostrara para los tipo de preguntas cerradas, en esta seccion podra adicionar nuevas respuestas y buscar respuestas creadas con anterioridad

 

11.    Botón nuevo.

Este botón habilita los controles (excepto el interruptor de estado) para crear una nueva pregunta. Si dichos controles tienen información, se limpian.

Al pulsarlo, se habilita el botón Cancelar y se deshabilitan el boton Editar. Este botón por defecto debe estar habilitado.

 

      12.    Botón editar.

      Este botón permite editar la pregunta seleccionada de la lista de datos y esta habilitado cuando se selecciona un registro de la lista. Se cargan los datos en la caja de texto y al pulsarlo el botón Editar, se habilitan todos los controles del formulario. Se habilitan los botones Guardar y Cancelar.

 

      13.  Botón guardar.

Este botón permite guardar una pregunta ya sea creada o modificada. Al pulsarlo se deshabilitan los botones Editar, Cancelar y el propio botón, y se habilita el boton Nuevo.

 

      14.  Botón cancelar.

Este botón cancela, limpia y deshabilita los controles del formulario incluido el interruptor del estado. Deshabilita los botones Editar y Guardar .

 

      15.  Listado de preguntas.

Es una lista de datos que contiene los campos de: nombre, tipo  y estado. Dicha lista puede ser ordenada y/o filtrada por cada campo. La fila seleccionada siempre se mostrará resaltada en azul claro. La fila inactiva se mostrará en rosado.
