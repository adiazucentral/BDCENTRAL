Integración de middleware por laboratorio

Este formulario permite relacionar una URL de servicio web para integrar el sistema Middleware con los laboratorios internos registrados en el sistema LIS.

A continuación, se muestra una ventana típica con sus partes:

 [Imagen 01 - Formulario de integración de middleware]

1. Buscador.
Se escribe o se introduce un valor alfa-numérico para filtrar elementos de la lista de datos de los laboratorios. El filtro se da de manera dinámica; a medida que se vaya digitando un valor, la lista va mostrando las filas coincidentes con las columnas visibles.

2. Ordenamiento.
Se usa el ordenamiento por columnas de menor a mayor o viceversa, la lista se organiza ya sea alfabéticamente, numéricamente o por valor (falso o verdadero). El icono cambia según el orden.

3. Cantidad.
Este control muestra la cantidad de laboratorios activos e internos visualizados en la lista de datos. Su valor cambia a medida que se vayan filtrando valores en el cuadro de búsqueda.

4. Botón ayuda. 
Despliegue del manual de cada página. 

5. Fecha de última actualización - Usuario.
Aquí se muestra la fecha y el usuario que modificaron el registro que ha sido seleccionado. Por defecto no aparece visible, solo cuando se selecciona una fila de la lista de laboratorios.

6. URL.  
Es la ruta de acceso al servicio web para integrar Middleware con el laboratorio seleccionado. Este campo es requerido.

7. Probar conexión.
Es un botón que valida la URL introducida, si la URL no es válida o falla la conexión aparece un mensaje en rojo indicando dicho inconveniente.

8. Enviar orden desde el ingreso de órdenes.
Indica si el LIS en el módulo de ingreso de órdenes envía la información de la orden para que sea recibida y almacenada por el Middleware. Se valida que al menos un tipo de envío deba estar activo.

9. Enviar orden desde la verificación de muestras.
Indica si el LIS en el módulo de verificación de muestras envía la información de la orden para que sea recibida y almacenada por el Middleware. Se valida que al menos un tipo de envío deba estar activo.

10. Botón editar.
Este botón permite editar la URL del laboratorio selecciono de la lista de datos; es habilitado cuando se selecciona un registro. Se cargan los datos en la caja de texto URL y al pulsarlo el botón Editar , se habilitan todos los controles del formulario. Se habilitan los botones Guardar y Cancelar.

11. Botón guardar.
Este botón permite guardar los cambios realizados en la URL y tipo de envío de orden del laboratorio seleccionado. Al pulsarlo se deshabilitan los botones Editar, Cancelar y el propio botón.

12. Botón cancelar.
Este botón cancela, limpia y deshabilita los controles del formulario. Deshabilita los botones Editar y Guardar.

13. Listado de laboratorios.
Es una lista de datos que contiene dos campos: Laboratorio que aparece con el código unido al nombre y la URL del servicio web. Dicha lista puede ser ordenada y/o filtrada por cada campo. La fila seleccionada siempre se mostrará resaltada en azul claro.
