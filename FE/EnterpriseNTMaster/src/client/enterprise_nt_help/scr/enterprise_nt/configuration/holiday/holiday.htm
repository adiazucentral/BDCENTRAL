Festivos

La asignación de festivos es la herramienta que provee la aplicación para definir los dias que son festivos.

A continuación, se muestra una ventana típica con sus partes: 


[Imagen de referencia -Asignación de festivos]

1. Años. 
Usa el control para seleccionar el año en que se van asignar los festivos, solo se cargan dos años el actual y el siguiente.  

2. Días inhabilitados.
Son todos los días anteriores al día actual, solo permite visualizar como se nombró el festivo, no permite editar.

3. Día en que se encuentra.
Se visualiza de color verde y corresponde al día en que se encuentra la aplicación, no permite editar.

4. Días que ya contiene un festivo creado.
Se visualiza de color  azul y ver como se nombró el festivo, se puede editar o eliminar, si desea editar o eliminar de click sobre un festivo creado y se generaría una imagen similar a la siguiente:

[Imagen de referencia - Modificación del festivo]

Para crear  un nuevo festivo debe dar click sobre los dias que no se encuentran inhabilitados con lo que se generaría una imagen similar a la siguiente:


[Imagen de referencia - Creación del festivo]

 

Se le da un nombre al evento y se guarda el festivo y se vizualiza de color azul.
