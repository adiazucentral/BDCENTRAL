Grupos etarios

El maestro de los grupos poblacionales define un nombre para establecer un rango mínimo y máximo de edades asociadas al género de cada paciente para generar estadísticas.

A continuación, se muestra una ventana típica con sus partes:

1.   

2.   1. Ordenamiento.

Se usa el ordenamiento por columnas de menor a mayor o viceversa, la lista se organiza ya sea alfabéticamente o por valor (falso o verdadero). El ícono que aparece en cada columna cambia según el orden.

 

2.Cuadro de texto de búsqueda.

Se escribe o se introduce un valor alfa-numérico para filtrar elementos de la lista de datos de los grupos poblacionales. El filtro se da de manera dinámica; a medida que se vaya digitando un valor, la lista va mostrando las filas coincidentes con las columnas visibles.

 

3.   3. Cantidad.

Este control muestra la cantidad de grupos poblacionales visualizados en la lista de datos. Su valor cambia a medida que se vayan filtrando valores en el cuadro de búsqueda o cuando se introduce un nuevo grupo poblacional.

 

4.   4. Fecha de última actualización - Usuario.                                                                                                       

Aquí se muestra la fecha y el usuario que modificó el grupo poblacional seleccionado. Por defecto no aparece visible, solo cuando se selecciona una fila de la lista de grupos poblacionales.

 

5.   5. Botón ayuda. 

Aquí se despliega el manual de cada página.

 

6.   6. Código .                                                                                                      

Código para identificar cada grupo no se puede repetir, es un campo alfa-numérico que acepta mayúsculas y minúsculas, Es un campo requerido. .

 

7. Nombre.                                                                                                      

Es un campo de texto donde se nombra el grupo poblacional para ser identificado. Es requerido y se valida si está repetido.

 

7.   8. Género.                                                                                                      

Es un campo donde se selecciona el género que filtrarán los pacientes, cuenta con 4 géneros, Todos, Masculino, Femenino e Indefinido.

 

9.Unidad de tiempo.                                                                                                      

Es un campo donde se selecciona la medida de tiempo para generar un rango. Si es en Años o Días .

 

 

9.   10.  Rango inicial.

Es un campo numérico incremental de 1 en 1 y que inicia dependiendo de la unidad de tiempo. Si es por años, inicia en 1, si es por días su valor mínimo es 0.

 

11. Rango final.

Es un campo numérico incremental de 1 en 1 y que inicia dependiendo de la unidad de tiempo. Si es por años, inicia en 200 como valor máximo, si es por días su valor máximo es 365.

 

      11.  Estado.

Es un campo en forma de  interruptor que permite activar o desactivar un grupo poblacional. Al cambiar el estado muestra un cuadro de confirmación donde pregunta si se encuentra seguro de cambiar el estado o no.

 

12.  Botón nuevo.

Este botón habilita las cajas de texto (excepto el interruptor de estado) para crear un nuevo grupo poblacional. Si dichas cajas de texto tienen datos, se limpian.Al pulsarlo, se habilita el botón Cancelar y se deshabilitan los botones Editar e Imprimir. Este botón por defecto debe estar habilitado.

 

13.  Botón editar.

Este botón permite editar el grupo poblacional seleccionado de la lista de datos y es habilitado cuando se selecciona de la lista. Se cargan los datos en las cajas de texto y al pulsar el botón Editar, se habilitan todos los controles del formulario. Se habilitan los botones Guardar y Cancelar, y se deshabilita el botón Imprimir.

 

      14. Botón guardar.

Este botón permite guardar un grupo poblacional ya sea creado o modificado. Al pulsarlo se deshabilitan los botones Editar, Cancelar y el propio botón, y se habilitan los botones Nuevo e Imprimir.

 

      15.  Botón cancelar.

Este botón cancela, devuelve y deshabilita las cajas de texto incluido el interruptor del estado. Deshabilita los botones Editar y Guardar, y habilita el botón Imprimir y Nuevo.

 

      16.  Botón Reporte

Este botón abre una venta del reporte de los grupos poblacionales que se encuentran en la lista de datos. Se muestra con los mismos registros filtrados y con la opción de exportar a Excel, PDF u otros formatos.

 

      17.  Listado de grupos poblacionales.

Es una lista de datos que contiene 4 campos: Nombre, edad, Género y Estado. Dicha lista puede ser ordenada y/o filtrada por cada campo. La fila seleccionada siempre se mostrará resaltada en azul claro. Las filas inactivas se mostraran resaltadas en rosado.
