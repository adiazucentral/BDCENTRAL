Configuración dependencia de demográficos

La configuración de dependencia de demográficos se crea con el fin de hacer más fácil la configuración en una instalación inicial de Enterprise NT
A continuación, se muestra una ventana típica con sus partes:


 

1. Usuario- fecha de última actualización.
Aquí se muestra la fecha y el usuario que modificaron el comentario que ha sido  seleccionado. Por defecto no aparece visible, solo cuando se selecciona una fila de la lista de comentarios.

2. Botón ayuda.    
Este botón despliega el manual de cada página.

3. Sede central.     
Esta opción aparecera si se encuentra habilitado el manejo de multisedes, y en este control se configura la sede que va ser la sede central de la base de datos.

4. Demográfico para excluir pruebas.
Demográfico por defecto que se utilizara para la configuración de las pruebas que no van a ser procesadas en los ítems del demográfico seleccionado.

5. Demográfico PyP 
Demográfico que se utilizara para la prevención y promoción de las pruebas  adicionadas a una orden según configuración adicional en los ítems del demográfico.

6. Demográfico para el título del informe.
Demográfico que se mostrara en el encabezado de los reportes de resultados, como así mismo controla el nombre de los reportes cuando se descarguen directamente o sean enviados por correo electrónico.

7. Ubicacíon hospitalaria.
Demográfico para configurar el demográfico que se va usar para la ubicación hospitalaria.

8. Demográfico para la encripción del reporte al enviar el correo.
Demográfico para configurar cuales de sus items encriptara el reporte de resultados.

9 Demográfico que valida inconsistencia.
Demográfico de la historia empleado para verificar las inconsistencias de los datos de la historia en las interfaces con sistemas centrales.

10. Indica si se usa demográfico adicional para la consulta web.
Permite habilitar la configuración de los usuarios para la consulta web.

11. Demográfico adicional para consulta web.
Demográfico para configurar el usuario y la contraseña a los item demográficos para loguearse en consulta web.

12. Contraseña genérica para consulta web.
 Contraseña generica para loguearse la primera vez en consulta web.  

13. Demográfico para el envio de correo automatico.
Demográfico para el envio de correo automatico cuando los examanes se validan.

14 Item para el envio de correo automatico.
Item demográfico para el envio de correo automatico cuando los examanes se validan

15 Demográfico de histograma.
Demográfico utilizado por defecto para la generación del reporte de histograma del LIS

16 Ítem demográfico de histograma.
Ítem de demográfico dependiente del demográfico de histograma que se utilizara para la generación de la gráfica en el reporte de histograma del LIS

17 Botón guardar.
Este botón permite guardar las modificaciones realizada en las llaves de configuración, mostrara un mensaje para confirmar la acción a realizar

18 Botón cancelar.
Este botón permite cancelar las modificaciones que se hayan realizado en el formulario y no desea guardar.
