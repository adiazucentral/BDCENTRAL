Patrones disponibles

Puedes hacer tu máscara usando algunos patrones disponibles. Si usa un patrón no especificado a continuación para construir su máscara, se considerará un divisor. Un divisor es un carácter usado para agrupar elementos semánticos, por ejemplo: "-" en SSN para separar el área, el grupo y los números de serie o "." en IPv4 para crear 4 grupos de 8 bits.

Patrones comunes
  • '*' => /./ Cualquier personaje individual
  • 'w' => / \ w / Cualquier carácter de palabra (letra, número, guión bajo)
  • 'W' => / \ W / Cualquier carácter no verbal
  • 'd' => / \ d / Cualquier dígito
  • 'D' => / \ D / Cualquier no dígito
  • 's' => / \ s / Cualquier carácter en blanco
  • 'S' => / \ S / Cualquier carácter que no sea de espacio en blanco
  • 'b' => / \ b / Cualquier límite de palabra
