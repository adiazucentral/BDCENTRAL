Items demográficos

El maestro de los ítems de demográficos hace referencia al detalle de cada uno de los demográficos codificados
A continuación, se muestra una ventana típica con sus partes:

1. Cuadro de texto de búsqueda.                                                                                          
Se escribe o se introduce un valor alfa-numérico para filtrar elementos de la lista de datos de los ítems demográficos. El filtro se da de manera dinámica; a medida que se vaya digitando un valor, la lista va mostrando las filas coincidentes con las columnas visibles.
 
2. Ordenamiento.
Se usa el ordenamiento por columnas de menor a mayor o viceversa, la lista se organiza ya sea alfabéticamente o por valor (falso o verdadero). El ícono que aparece en cada columna cambia según el orden.
 
3. Cantidad.
Este control muestra la cantidad de ítems demográficos visualizados en la lista de datos. Su valor cambia a medida que se vayan filtrando valores en el cuadro de búsqueda o cuando se introduce un nuevo ítem demográfico. 
 
4. Fecha de última actualización - Usuario.                                                                                                      
Aquí se muestra la fecha y el usuario que modificó el ítem demográfico seleccionado. Por defecto no aparece visible, solo cuando se selecciona una fila de la lista de datos.
 
5. Botón ayuda.   
Aquí se despliega el manual de cada página.  
 
6. Origen.                                                                                                       
Este campo es para filtrar los demográficos si es por orden o por historia.

7. Demográficos.                                                                                                       
Es un campo de selección donde aparece solo la lista de demográficos codificados activos listos para asignarles ítems.  
 
8. Código.                                                                                                       
Es un campo que identifica cada uno de los ítems, del detalle del demográfico.
Este código es único para cada demográfico, es decir se puede repetir, pero con diferente padre. 
 
9. Nombre.                                                                                                       
Es un campo alfa-numérico que nombra al ítem demográfico para referenciarlo en una lista tanto en ingreso de órdenes como en la historia clínica de un paciente, acepta mayúsculas y minúsculas; lo máximo son 60 caracteres y es un dato que siempre será requerido. 
 
10. Observaciones.
Es un campo multi-lineal que almacena información adicional y que se pueda requerir o imprimir.
 
11. Ítem por defecto.
Es un campo en forma de interruptor que indica si el ítem demográfico es el que se mostrará por defecto cuando se cree una nueva orden. Solo puede existir uno marcado por demográfico
 
12. Estado.
Es un campo que permite activar o desactivar un ítem demográfico. Al cambiar el estado muestra un cuadro de confirmación donde pregunta si se encuentra seguro de cambiar el estado o no. El control tiene forma de interruptor (switch).
 
13. Botón nuevo.
Este botón habilita las cajas de texto (excepto el interruptor de estado) para crear un nuevo ítem demográfico. Si dichas cajas de texto tienen datos, se limpian.
Al pulsarlo, se habilita el botón Cancelar y se deshabilitan los botones Editar e Imprimir . Este botón por defecto debe estar habilitado.
 
14.  Botón editar.
Este botón permite editar el ítem demográfico seleccionado de la lista de datos y es habilitado cuando se selecciona de la lista. Se cargan los datos en las cajas de texto y al pulsar el botón Editar, se habilitan todos los controles del formulario. Se habilitan los botones Guardar y Cancelar, y se deshabilita el botón Imprimir.
 
15. Botón guardar.
Este botón permite guardar un ítem demográfico ya sea creado o modificado. Al pulsarlo se deshabilitan los botones Editar, Cancelar y el propio botón, y se habilitan los botones Nuevo e Imprimir.
 
16. Botón cancelar.
Este botón cancela, devuelve y deshabilita las cajas de texto incluido el interruptor del estado. Deshabilita los botones Editar y Guardar, y habilita el botón Imprimir y Nuevo.
 
17. Botón reporte.
Este botón abre una ventana donde se muestra el informe de los ítems demográficos que se encuentran en la lista de datos. Se muestra los mismos registros filtrados y las opciones para guardar, ya sea en PDF o en Excel.
 
18. Listado de ítems demográficos.
Es una lista de datos que contiene 4 campos: Código, Nombre, Ítem por defecto y Estado. Dicha lista puede ser ordenada y/o filtrada por cada campo. La fila seleccionada siempre se mostrará resaltada en azul claro. Las filas inactivas se mostraran resaltadas en rosado.
