Pruebas por laboratorio

Este maestro asigna las pruebas a un laboratorio, ya sea interno o externo, teniendo en cuenta las sedes y el tipo de orden.

A continuación, se muestra una ventana típica con sus partes:

      1. Cuadro de texto de búsqueda de laboratorios.

Se escribe o se introduce un valor alfa-numérico para filtrar elementos de la lista de datos de los laboratorios. El filtro se da de manera dinámica; a medida que se vaya digitando un valor, la lista va mostrando las filas coincidentes con las columnas visibles.

 

2. Lista de seleccion de sedes.

Lista las sedes activas del sistema.

 

3. Ordenamiento de laboratorios.

Se usa el ordenamiento por columnas de menor a mayor o viceversa, la lista se organiza ya sea alfabéticamente, numéricamente o por valor (falso o verdadero). El ícono que aparece en cada columna cambia según el orden.

 

      4.   Cantidad de sedes. 

Este control muestra la cantidad de registros visualizados en la lista de seleccion de sedes.

 

5.   Cantidad de laboratorios. 

Este control muestra la cantidad de registros visualizados en la lista de laboratorios. Su valor cambia a medida que se vayan filtrando valores en el cuadro de búsqueda.

 

6. Cuadro de texto de búsqueda de pruebas.

Se escribe o se introduce un valor alfa-numérico para filtrar elementos de la lista de datos de las pruebas. El filtro se da de manera dinámica; a medida que se vaya digitando un valor, la lista va mostrando las filas coincidentes con las columnas visibles.

 

7. Ordenamiento de pruebas.

Se usa el ordenamiento por columnas de menor a mayor o viceversa, la lista se organiza ya sea alfabéticamente, numéricamente o por valor (falso o verdadero). El ícono que aparece en cada columna cambia según el orden.

 

      8. Rutina.

Columna que visualizar las pruebas asignadas, no asignadas o bloqueadas para los tipos de ordenes considerados como rutinas, re-llamdos, entre otros y el laboratorio seleccionado.

 

9. Botón ayuda. 

Aquí se despliega el manual de cada página.

 

      10. Urgencia.

Columna que visualizar las pruebas asignadas, no asignadas o bloqueadas para los tipos de ordenes considerados como urgencias, el laboratorio y sede seleccionados.

 

      11 . Todos.                                                                                                      

Fila para seleccionar todas las rutinas y/o urgencias de las pruebas y/o perfiles activos del sistema en el laboratorio y sede seleccionados .

 

     

      12. Icono de candado.                                                                                                        

Icono informativo para advertir que la prueba ya se encuentra asignada en otro laboratorio para los tipos de orden de urgencia o de rutina, al dar click sobre el saldrá un mensaje informado en que laboratorio está asignada esa prueba

.

 

13. Icono de selección   .                                                                                                      

Icono para seleccionar una prueba en el tipo de orden de rutina o urgencia y asignarla a un laboratorio y una sede

 

14. Icono de no selección   .                                                                                                      

Icono para deseleccionar una prueba en el tipo de orden de rutina o urgencia y desasignarla de un laboratorio y una sede

 

      15.  Botón guardar.

Este botón permite guardar las pruebas asignadas a un laboratorio y sede seleccionados.

 

      16.  Botón cancelar.

Este botón cancela, todos los cambios no deseados que se hayan hecho en el formulario.

 

      17.  Botón reporte

Este botón abre una ventana donde se muestra una previsualizacion del reporte, en este se podra escoger en que formato se desea exportar el listado de las pruebas asignadas a un laboratorio y una sede, este informe debe ser acorde al filtro y ordenamiento escogido en la tabla de datos del formulario inicial.

 

18.Listado de laboratorios.

Es una lista de datos que contiene dos campos: Código, labratorio y tipo. Dicha lista puede ser ordenada y/o filtrada por cada campo. La fila seleccionada siempre se mostrará resaltada en azul claro. La fila inactiva se mostrará en rosado.
