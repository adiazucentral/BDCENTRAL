Agente Etiológico

Esta opción me permite realizar una búsqueda de microorganismos para la obtención de estadísticas con el objetivo de reportar la incidencia de estos a la secretaria de salud de cada departamento.

A continuación, se muestra una ventana típica con sus partes:

 

[Imagen 01 - Formulario de agentes etiológicos]

 

1. Buscador.
Escribe una palabra o frase en el cuadro de búsqueda para buscar un elemento. Dado que la búsqueda se inicia en cuanto empiezas a escribir, si por ejemplo escribe "B", todos los agentes etiológicos cuyo valor empiece por la letra B se mostrarán en la lista.

2. Ordenamiento.
Usa el control de ordenamiento para ordenar de menor a mayor o viceversa según el campo que desees ordenar solo al hacer click  sobre él.  

3. Cantidad.
Este control muestra la cantidad de agentes etiológicos visualizados en la lista.

4. Fecha - Usuario de última actualización.
Se despliega la fecha y el usuario que modificaron el agente etiológico que ha sido seleccionado.

5. Botón ayuda.
Aquí despliega la explicación de cada uno de los controles

6. Buscar por. 
Seleccionar en que parte del módulo de microbiología va encontrarte el agente si por comentario o por antibiograma.

7. Microorganismo.
Nombre del microorganismo que se va a emplear

8. Código del grupo 
Código del grupo de microorganismo al que se vaya a pertenecer.

9. Clasificación.
Analizador en el cual será procesada la muestra.

10. Botón nuevo.
Este botón habilita las cajas de texto para crear un nuevo agente etiológico. Si dichas cajas de texto tienen datos, se limpian. Al pulsarlo, se habilita el botón Cancelar y se deshabilitan los botones Editar y Reporte . Este botón por defecto está habilitado.

11. Botón editar.
Este botón permite editar un agente etiológico seleccionado de la lista de datos y es habilitado cuando se selecciona un agente etiológico de la lista. Se cargan los datos en las cajas de texto y al pulsarlo el botón Editar, se habilitan todos los controles del formulario. Se habilitan los botones Guardar y Cancelar, y se deshabilita el botón Reporte.

12. Botón guardar.
Este botón permite guardar un agente etiológico ya sea creado o modificado. Al pulsarlo se deshabilitan los botones Editar, Cancelar y el propio botón, y se habilitan los botón Nuevo y Reporte.

13. Botón cancelar.
Este botón cancela, limpia y deshabilita las cajas de texto. Deshabilita los botones Editar y Guardar, y habilita el botón Reporte.

14. Botón  Reporte.
Este botón imprime los agentes etiológicos que se encuentran en la lista de datos. Se muestra un reporte con las mismas columnas y los mismos registros filtrados.

15. Listado de agentes etiológicos
Es una lista de datos que contiene los datos de: Buscar por, microorganismo, código del grupo y clasificación. Dicha lista puede ser ordenada y/o filtrada por cada campo. La fila seleccionada siempre se mostrará resaltada en azul claro.
