Banco

Este maestro define el nombre de los bancos que se registrarán en el módulo de caja y facturación.

A continuación, se muestra una ventana típica con sus partes:

     

      1.  Ordenamiento.

Se usa el ordenamiento por columnas de menor a mayor o viceversa, la lista se organiza ya sea alfabéticamente, numéricamente. El ícono que aparece en cada columna cambia según el orden.

 

      2.  Cuadro de búsqueda.                                                                                          

Se escribe o se introduce un valor alfa-numérico para filtrar elementos de la lista de bancos. El filtro se da de manera dinámica; a medida que se vaya digitando un valor, la lista va mostrando las filas coincidentes con las columnas visibles.

 

      3.  Cantidad. 

Este control muestra la cantidad de bancos visualizados en la lista de datos. Su valor cambia a medida que se vayan filtrando valores en el cuadro de búsqueda o cuando se introduce un nuevo banco.

 

      4.  Usuario- fecha de última actualización.                                                                                                     

Aquí se muestra la fecha y el usuario que modificaron el banco que  ha sido  seleccionado. Por defecto no aparece visible, solo cuando se selecciona una  fila de la lista de bancos. 

 

      5.  Botón ayuda.

Aquí se despliega el manual de cada página

 

6.  Banco.

      Nombre del banco a definir.el cual no se puede repetir Es un campo alfa-numérico que acepta mayúsculas y minúsculas, lo máximo son 60 caracteres, Es un campo requerido.

 

      7.  Estado.

Este botón Indica si registro está activo o no, al modificar el estado siempre hay aparece un cuadro de dialogo que pide la confirmación.

 

      8.  Botón nuevo.

Este botón habilita las cajas de texto (excepto el interruptor de estado) para crear un nuevo banco.  Si dichas cajas de texto tienen datos, se limpian. Al pulsarlo, se habilita el botón Cancelar y se deshabilitan los botones Editar e Imprimir. Este botón por defecto debe estar habilitado.

 

      9.  Botón editar.

Este botón permite editar el banco  seleccionado de la lista de datos y es habilitado cuando se selecciona un banco  de la lista. Se cargan los datos en las cajas de texto y al pulsarlo el botón Editar, se habilitan todos los controles del formulario. Se habilitan los botones Guardar y Cancelar, y se deshabilita el botón Imprimir.

 

      10.  Botón guardar.

Este botón permite guardar un banco ya sea creada o modificada. Al pulsarlo se deshabilitan los botones Editar, Cancelar y el propio botón, y se habilitan los botones Nuevo e Imprimir. 

 

      11.  Botón cancelar.

Este botón cancela, limpia y deshabilita las cajas de texto incluido el interruptor del estado. Deshabilita los botones Editar y Guardar, y habilita el botón Imprimir.

 

      12.  Botón reporte.

Este botón imprime los bancos que se encuentran en la lista de datos. Se muestra un reporte con las mismas columnas y los mismos registros filtrados.   

  

      13.  Lista de bancos.

Es una lista de datos que contiene dos campos: banco y Estado. Dicha lista puede ser ordenada y/o filtrada por cada campo. La fila seleccionada siempre se mostrará resaltada en azul claro y las neveras inactivas estarán resaltadas con rosado.
