Método de recolección

Este maestro define los metodos de recoleccción para las muestras de  micro biología.

A continuación, se muestra una ventana típica con sus partes:

     

      1.    Ordenamiento.

Se usa el ordenamiento por columnas de menor a mayor o viceversa, la lista se organiza ya sea alfabéticamente, numéricamente o por valor (falso o verdadero). El ícono que aparece en cada columna cambia según el orden.

 

1.     Cuadro de texto de búsqueda. 

       Se escribe o se introduce un valor alfa-numérico para filtrar elementos de la lista de datos. El filtro se da de manera dinámica; a medida que se vaya digitando un valor, la lista va mostrando las filas coincidentes con las columnas visibles.

3.    Cantidad.

Este control muestra la cantidad de registros visualizados en la lista de datos. Su valor cambia a medida que se vayan filtrando valores en el cuadro de búsqueda o cuando se introduce una nueva tarjeta de crédito.

    

      4.    Fecha de última actualización - Usuario.                                                                                                     

Aquí se muestra la fecha y el usuario que creó o modificó el registro de método de recolección microbiología que ha sido seleccionada. Por defecto no aparece visible, solo cuando se selecciona una fila de la lista de datos.

 

      5.    Botón ayuda. 

Aquí se despliega el manual de cada página.

 

 

      6.    Método de recolección.                                                                                                      

Es un campo alfa-numérico que describe el método de recolección de microbiología, acepta mayúsculas y minúsculas; lo máximo son 60 caracteres y es un dato que siempre será requerido y validado en caso de estar repetido.

 

 

      7.    Estado.

Indica si el método de recolección se encuentra activa o no. Al cambiar el estado muestra un cuadro de confirmación donde pregunta si se encuentra seguro de cambiar el estado o no. El control tiene forma de interruptor (switch).

 

 

      8.    Botón nuevo.

Este botón habilita los controles (excepto el interruptor de estado) para crear un nuevo registro. Si dichos controles tienen información, se limpian.

Al pulsarlo, se habilita el botón Cancelar y se deshabilitan los botones Editar e Imprimir. Este botón por defecto debe estar habilitado.

 

 

      9.    Botón editar.

Este botón permite editar el método de recolección seleccionado de la lista de datos y es habilitado cuando se selecciona un registro de la lista. Se cargan los datos en la caja de texto y al pulsarlo el botón Editar, se habilitan todos los controles del formulario. Se habilitan los botones Guardar y Cancelar, y se deshabilita el botón Imprimir.

 

 

      10.  Botón guardar.

Este botón permite guardar un método de recolección ya sea creada o modificada. Al pulsarlo se deshabilitan los botones Editar, Cancelar y el propio botón, y se habilitan los botones Nuevo e Imprimir.

 

 

      11.  Botón cancelar.

Este botón cancela, limpia y deshabilita los controles del formulario incluido el interruptor del estado. Deshabilita los botones Editar y Guardar, y habilita el botón Imprimir.

 

 

      12.  Botón reporte

Este botón abre una ventana donde se muestra el reporte de los métodos de recolección de microbiología que se encuentran en la lista de datos. Se muestra los mismos registros filtrados y las opciones para guardar ya sea en PDF o en Excel.

 

 

      13.   Listado de tareas de microbiología.

Es una lista de datos que contiene dos campos: Método de recolección y Estado. Dicha lista puede ser ordenada y/o filtrada por cada campo. La fila seleccionada siempre se mostrará resaltada en azul claro. La fila inactiva se mostrará en rosado.
