Exámenes por demográficos

Este maestro permite relacionar exámenes a los demográficos configurados en la seccion de ingreso de ordenes en la configuración de instalación.

A continuación, se muestra una ventana típica con sus partes:

[Imagen 01 - Formulario de exámenes por demográficos]

 

1. Buscador.
Escribe una palabra o frase en el cuadro de búsqueda para buscar un elemento. Dado que la búsqueda se inicia en cuanto empiezas a escribir, si por ejemplo escribe "B", todos los demograficos cuyo valor empiece por la letra B se mostrarán en la lista.

2. Ordenamiento.
Usa el control de ordenamiento para ordenar de menor a mayor o viceversa según el campo que desees ordenar solo al hacer click  sobre él.  

3. Cantidad.
Este control muestra la cantidad de relaciones exámenes - demográficos visualizadas en la lista.

4. Fecha - Usuario de última actualización.
Se despliega la fecha y el usuario que modificaron la relación exámenes - demográficos que ha sido seleccionada.

5. Botón ayuda.
Aquí despliega la explicación de cada uno de los controles

6. Demográfico #1.
Identificador único que identifica el demográfico #1 configurado, siempre es requerido.

7. Demográfico #2.
Identificador único que identifica el demográfico #2 configurado, siempre es requerido.

8. Demográfico #3.
Identificador único que identifica el demográfico #3 configurado, siempre es requerido.

9. Pruebas.
Esta opción despliega la lista de pruenas a los cuales se puede asociar a los demográficos. Esta opción es requerida, como mínimo debe existir una pruebas asociada.

10. Botón nuevo.
Este botón habilita las cajas de texto (excepto el interruptor de estado) para crear una nueva relación exámenes - demográficos. Si dichas cajas de texto tienen datos, se limpian. Al pulsarlo, se habilita el botón Cancelar y se deshabilitan los botones Editar y Reporte . Este botón por defecto está habilitado.

11. Botón editar.
Este botón permite editar la relación exámenes - demográficos seleccionada de la lista de datos y es habilitado cuando se selecciona una relación exámenes - demográficos de la lista. Se cargan los datos en las cajas de texto y al pulsarlo el botón Editar, se habilitan todos los controles del formulario. Se habilitan los botones Guardar y Cancelar, y se deshabilita el botón Reporte.

12. Botón guardar.
Este botón permite guardar una relación exámenes - demográficos ya sea creada o modificada. Al pulsarlo se deshabilitan los botones Editar, Cancelar y el propio botón, y se habilitan los botón Nuevo y Reporte.

13. Botón cancelar.
Este botón cancela, limpia y deshabilita las cajas de texto incluido el interruptor del estado. Deshabilita los botones Editar y Guardar, y habilita el botón Reporte.

14. Botón  Reporte.
Este botón imprime las relaciones exámenes - demográficos que se encuentran en la lista de datos. Se muestra un reporte con las mismas columnas y los mismos registros filtrados.

15. Listado de relaciones exámenes - demográficos.
Es una lista de datos que contiene los valores de los 3 demográficos configurados. Dicha lista puede ser ordenada y/o filtrada por cada campo. La fila seleccionada siempre se mostrará resaltada en azul claro.
