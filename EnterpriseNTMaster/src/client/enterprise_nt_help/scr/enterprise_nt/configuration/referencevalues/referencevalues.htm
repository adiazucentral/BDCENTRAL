Valores de referencia

Los valores de referencia definen un rango de aceptación en el que se considera que el resultado es normal para el paciente.

A continuación, se muestra una ventana típica con sus partes:

      1.    Cuadro de texto de búsqueda de pruebas.

Se escribe o se introduce un valor alfa-numérico para filtrar elementos de la lista de datos de las pruebas. El filtro se da de manera dinámica; a medida que se vaya digitando un valor, la lista va mostrando las filas coincidentes con las columnas visibles.

 

2.    Lista de seleccion de areas.

Lista las areas activas del sistema.

 

3.    Cantidad de areas. 

Este control muestra la cantidad de registros visualizados en la lista de seleccion de areas.

 

4.  Lista de valores de referencia por prueba .

Area para visualizar los valores de referencia asignados a cada prueba, esta podra ser visualizada de dos maneras:

4.1 PRUEBAS TIPO TEXTO

 

 

Para estas pruebas, se mostran las columnas de genero, resultado literal, resultado literal pánico y el comentario (si el sistema tiene la llave de menejo raza activa, este campo tambien sera visible).

 

Para editar un registro, se tiene que dar doble click sobre el, si se quiere agregar un nuevo valor de referencia de click en el botón nuevo, cualquiera de estas acciones generan una ventana emergente como la siguiente

 

4.1.1. Género: Género al que aplica el valor de referencia.

4.1.2 Raza: Raza para que se aplicara el valor de referencia.

4.1.3 Resultado literal: Valor que representa que el resultado tiene un valor normal .

4.1.4. Resulado literal pánico: Valor que representa que el resultado tiene un valor anormal.

4.1.5. Notificación: Valor que representa si la abra notificación del valor de referencia o nop.

4.1.6. Comentario: Comentario que se imprime en el reporte de resultados cuando se toma el valor de referencia correspondiente. Este texto complementa el valor de referencia.

4.1.7 . Botón guardar: guardar los cambios realizado.

4.1.8. Botón cerrar: cierra la ventana emergente y no realizar ningun cambio.

 

 

4.2 PRUEBAS TIPO NUMERICO  

 

 

 

 

 

Para estas pruebas, se mostran los campos de género, edad minima, edad maxima, unidad de tiempo, rango minimo y rango maximo si el sistema tiene la llave de menejo raza activa, este campo tambien sera visible.

 

Para editar un registro, se tiene que dar doble click sobre el, si se quiere agregar un nuevo valor de referencia de click en el botón nuevo, cualquiera de estas acciones generan una ventana emergente como la siguiente:

 

 

4.2.1. Género: Género al que aplica el valor de referencia.

4.2.2. Unidad de tiempo: Puede ser días o años.

4.2.3. Edad mínima: Valor numérico entero que indica la edad mínima en que empieza a aplicar el valor definido para la referencia. Puede dejarse en blanco para los casos en que el valor de referencia indica solo el valor máximo con esto el valor por defecto lo marcará la unidad.

4.2.4. Norma mínima: Valor de la referencia descrito en el inserto (documento ditribuido por proveedor de los reactivos, que indica los valores de referencia para cada examen). Puede quedar en blanco si solo se especifica valor máximo. Los decimales del valor registrado se toman a partir del campo “decimales” del maestro de exámenes

4.2.5. pánico mínimo: Valor inferior al valor minimo de referencia, el cual indica alarma visual en registro de resultados. El registro de este valor no es obligatorio pero cuando se ingrese debe existir máximo y mínimo.

4.2.6 Reportable mínimo: Valor mínimo que se puede reportar, para validar el ingreso de valores incongruentes a un resultadio real.

4.2.7. Notificación: Valor que representa si la abra notificación del valor de referencia o nop.

4.2.9 Raza: Raza para que se aplicara el valor de referencia.

4.2.10. Critico: Indica si el valor de pánico es critico o no.

4.2.11. Edad máxima: Valor numérico entero que indica la edad máxima en que termina el valor definido para la referencia. Para unidad de tiempo Años se cuenta hasta 200, para días hasta 365. Puede dejarse en blanco para los valores en que se indique solo edades minimas con esto el valor por defecto es 200 para años y 365 para días.

4.2.12. Norma máxima: Valor de la referencia descrito en el inserto (documento ditribuido por proveedor de los reactivos, que indica los valores de referencia para cada examen). Puede quedar en blanco si solo se especifica valor mínimo. Los decimales del valor registrado se toman a partir del campo “decimales” del maestro de exámenes.

4.2.10. Pánico máximo: Valor superior al valor máximo de referencia, el cual indica alarma visual en registro de resultados. El registro de este valor no es obligatorio pero cuando se ingrese debe existir máximo y mínimo.

4.2.11. Reportable máximo: Valor máximo que se puede reportar para validar el ingreso de valores incongruentes con un resultado real.

4.2.12. Comentario: Comentario que se imprime en el reporte de resultados cuando se toma el valor de referencia correspondiente. Este texto complementa el valor de referencia.

4.2.13. Botón guardar: guardar los cambios realizado.

4.2.14. Botón cerrar: cierra la ventana emergente y no realizar ningun cambio.

 

5.     Botón ayuda. 

Aquí se despliega el manual de cada página.

 

      6.   Tipo .

Icono informativo para distinguir el tipo de exámen.

 

7.  Procesamiento .

Icono informativo para el procesamiento del exámen

 

8.  Botón nuevo.

Botón para agregar un nuevo valor de referencia en el examen que este seleccionado.

    

      9.  Botón reporte

Este botón abre una ventana donde se muestra una previsualizacion del reporte, en este se podra escoger en que formato se desea exportar el listado de los valores de referencia asociados a una prueba.

 

10.  Listado de examenes.

Es una lista de datos que contiene dos campos: Código, nombre y tipo. Dicha lista puede ser ordenada y/o filtrada por cada campo. La fila seleccionada siempre se mostrará resaltada en azul claro.
