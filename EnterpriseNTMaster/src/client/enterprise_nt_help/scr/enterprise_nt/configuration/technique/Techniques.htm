Tecnicas

El maestro de técnicas hace referencia a la forma o método en que se procesa un examen para la consecución del resultado.

A continuación, se muestra una ventana típica con sus partes:


[Imagen 01 - Formulario de técnicas]

1. Buscador.
Se escribe o se introduce un valor alfa-numérico para filtrar elementos de la lista de datos de las técnicas. El filtro se da de manera dinámica; a medida que se vaya digitando un valor, la lista va mostrando las filas coincidentes con las columnas visibles.

2. Ordenamiento.
Se usa el ordenamiento por columnas de menor a mayor o viceversa, la lista se organiza ya sea alfabéticamente, numéricamente o por valor (falso o verdadero). El icono cambia según el orden.

3. Cantidad.
Este control muestra la cantidad de técnicas visualizadas en la lista de datos. Su valor cambia a medida que se vayan filtrando valores en el cuadro de búsqueda o cuando se introduce una nueva técnica.

4. Botón ayuda. 
Despliegue del manual de cada página. 

5. Fecha de última actualización - Usuario.
Aquí se muestra la fecha y el usuario que modificaron la técnica que ha sido  seleccionada. Por defecto no aparece visible, solo cuando se selecciona una fila de la lista de técnicas.

6. Código.
Identificador único de la técnica. Es un campo alfa-numérico que acepta mayúsculas y minúsculas; lo máximo son 8 caracteres y es un dato que siempre será requerido.

7. Descripción.
Es la definición de la técnica que se emplea, Es un campo alfa-numérico que acepta mayúsculas, minúsculas y texto enriquecido, además, de ser multi-lineal. Es un campo requerido.

8. Estado.
Indica si la técnica se encuentra activa o no. Al cambiar el estado muestra un cuadro de confirmación donde pregunta si se encuentra seguro de cambiar el estado o no. El control tiene forma de interruptor (swich).

9. Botón nuevo.
Este botón habilita las cajas de texto (excepto el interruptor de estado) para crear una nueva técnica. Si dichas cajas de texto tienen datos, se limpian. Al pulsarlo, se habilita el botón Cancelar y se deshabilitan los botones Editar y Reporte. Este botón por defecto está habilitado.

10. Botón editar.
Este botón permite editar la técnica selecciona de la lista de datos y es habilitado cuando se selecciona una técnica de la lista. Se cargan los datos en las cajas de texto y al pulsarlo el botón Editar, se habilitan todos los controles del formulario. Se habilitan los botones Guardar y Cancelar, y se deshabilita el botón Reporte.

11. Botón guardar.
Este botón permite guardar una técnica ya sea creada o modificada. Al pulsarlo se deshabilitan los botones Editar, Cancelar y el propio botón, y se habilitan los botón Nuevo y Reporte.

12. Botón cancelar.
Este botón cancela, limpia y deshabilita las cajas de texto incluido el interruptor del estado. Deshabilita los botones Editar y Guardar, y habilita el botón Reporte.

13. Botón Reporte
Este botón imprime las técnicas que se encuentran en la lista de datos. Se muestra un reporte con las mismas columnas y los mismos registros filtrados.

14. Listado de técnicas.
Es una lista de datos que contiene tres campos: Código, Descripción y Estado. Dicha lista puede ser ordenada y/o filtrada por cada campo. La fila seleccionada siempre se mostrará resaltada en azul claro.
