Grupos

Este formulario define una agrupación de pruebas para facilitar el ingreso de ordenes y  filtrar listados y consultas.
A continuación, se muestra una ventana típica con sus partes:

   

1. Ordenamiento.
Se usa el ordenamiento por columnas de menor a mayor o viceversa, la lista se organiza ya sea alfabéticamente, numéricamente. El ícono que aparece en cada columna cambia según el orden.

2. Cuadro de búsqueda.                                                                                           
Se escribe o se introduce un valor alfa-numérico para filtrar elementos de la lista de grupos. El filtro se da de manera dinámica; a medida que se vaya digitando un valor, la lista va mostrando las filas coincidentes con las columnas visibles.

3. Cantidad.
Este control muestra la cantidad de grupos visualizados en la lista de datos. Su valor cambia a medida que se vayan filtrando valores en el cuadro de búsqueda o cuando se introduce una nuevo grupo. 

4. Usuario- fecha de última actualización.       
Aquí se muestra la fecha y el usuario que modificaron el grupo que ha sido  seleccionado. Por defecto no aparece visible, solo cuando se selecciona una  fila de la lista de grupos.

5. Botón ayuda.
Aquí se despliega el manual de cada página. 

6. Código.
Código para identificar cada grupo no se puede repetir, es un campo alfa-numérico que acepta mayúsculas y minúsculas, lo máximo son 60 caracteres, Es un campo requerido.

7. Grupo.
Nombre del grupo a definir. el cual no se puede repetir, es un campo alfa-numérico que acepta mayúsculas y minúsculas, lo máximo son 60 caracteres, Es un campo requerido.

8. Perfil.
Indica  si se toma en encabezados de los perfiles o el contenido.

9. Estado.
Este botón Indica si registro está activo o no, al modificar el estado siempre hay aparece un cuadro de dialogo que pide la confirmación.

10. Pruebas.
Este indica un listado de pruebas para asociar al grupo, según el botón de perfiles si está habilitado cargan las pruebas y los perfiles si esta deshabilitado solo cargan las pruebas.

11. Botón nuevo.
Este botón habilita las cajas de texto (excepto el interruptor de estado) para crear un nuevo grupo. Si dichas cajas de texto tienen datos, se limpian. Al pulsarlo, se habilita el botón Cancelar y se deshabilitan los botones Editar e Imprimir. Este botón por defecto debe estar habilitado.

12. Botón editar.
Este botón permite editar el grupo seleccionado de la lista de datos y es habilitado cuando se selecciona un grupo de la lista. Se cargan los datos en las cajas de texto y al pulsarlo el botón Editar, se habilitan todos los controles del formulario. Se habilitan los botones Guardar y Cancelar, y se deshabilita el botón Imprimir.

13. Botón guardar.
Este botón permite guardar un grupo ya sea creada o modificada. Al pulsarlo se deshabilitan los botones Editar, Cancelar y el propio botón, y se habilitan los botones Nuevo e Imprimir. 

14. Botón cancelar.
Este botón cancela, limpia y deshabilita las cajas de texto incluido el interruptor del estado. Deshabilita los botones Editar y Guardar, y habilita el botón Imprimir.

15. Botón reporte.
Este botón abre una ventana emergente que muestra el informe de grupos, permite exportar en PDF, Excel o imprimir los grupos que se encuentran en la lista de datos con las mismas columnas y los mismos registros filtrados.

16. Lista de grupos.
Es una lista de datos que contiene tres campos: Grupo y Estado. Dicha lista puede ser ordenada y/o filtrada por cada campo. La fila seleccionada siempre se mostrará resaltada en azul claro y los grupos inactivas estarán resaltados con rosado.
