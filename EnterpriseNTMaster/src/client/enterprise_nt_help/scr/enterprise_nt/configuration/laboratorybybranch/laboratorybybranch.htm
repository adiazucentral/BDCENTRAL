Laboratorio por sede

Este maestro permite asignar los laboratorios que estaran asociados a las sedes para permitir la verificación de las muestras.

A continuación, se muestra una ventana típica con sus partes:

      1.  Ordenamiento.

Se usa el ordenamiento por columnas de menor a mayor o viceversa, la lista se organiza ya sea alfabéticamente, numéricamente. El ícono que aparece en cada columna cambia según el orden.

 

      2.  Cuadro de búsqueda.                                                                                          

Se escribe o se introduce un valor alfa-numérico para filtrar elementos de la lista de sedes o laboratorios. El filtro se da de manera dinámica; a medida que se vaya digitando un valor, la lista va mostrando las filas coincidentes con las columnas visibles.

 

      3.  Cantidad.  

Este control muestra la cantidad de datos visualizados en la lista. Su valor cambia a medida que se vayan filtrando valores en el cuadro de búsqueda .

 

      4.  Usuario- fecha de última actualización.                                                                                                    

Aquí se muestra la fecha y el usuario que modificaron la sede que ha sido  seleccionado. Por defecto no aparece visible, solo cuando se selecciona una  fila de la lista. 

 

      5.  Botón ayuda.

Aquí se despliega el manual de cada página.

 

      6.   Lista de resultados literales.

Es una lista de datos que contiene dos campos: Laboratorio y asignado . Dicha lista puede ser ordenada y/o filtrada por cada campo. el campo asignado se selecciona cuando se quieren asignar los laboratorios  a la  sedes que se seleciono en la lista de sedes.

 

      7.  Botón guardar.

Este botón permite guardar los laboratorios asociados. Al pulsarlo se deshabilitan los botón  Cancelar y el propio botón, y se habilitan el  botón  Reporte.

 

      8.  Botón cancelar.

Este botón cancela, se devuelve al estado anterior antes de modificar el formulario. Deshabilita los botón Guardar, y habilita el botón Reporte.

 

      9.   Botón Reporte.

Este botón imprime una lista de laboratorios que se encuentran asociadas y no asociadas a la sede que selecciono.

 

      10.  Lista de pruebas.

Es una lista de datos que contiene dos campos: código y sede. Dicha lista puede ser ordenada y/o filtrada por cada campo. La fila seleccionada siempre se mostrará resaltada en azul claro.
