Pruebas por demográfico PyP

Este formulario define el filtro para las pruebas, por demográfico y rango de edades que se empleará al momento de generar el reporte de PyP.

 

A continuación se muestra la ventana especial para la asignación de las pruebas para un ítem demográfico seleccionado:

1. Ordenamiento.

Se usa el ordenamiento por columnas de menor a mayor o viceversa, ambas listas se organizan ya sea alfabéticamente o por valor (falso o verdadero) . El ícono que aparece en cada columna cambia según el orden.

 

2. Cuadros de texto de búsqueda.
Se escribe o se introduce un valor alfa-numérico para filtrar elementos de la lista de los item demográficos ubicada al lado izquierdo y se realiza la misma operación para el caso de la lista de pruebas ubicada a la derecha. El filtro se da de manera dinámica; a medida que se vayan digitando caracteres, la lista va mostrando las filas coincidentes con las columnas visibles. 

3. Nombre del demográfico configurado.
En esta caja de texto deshabilitada aparece el nombre del demográfico PyP que se encuentra configurado desde el módulo de instalación. 

4. Cantidad.

Estos controles muestran la cantidad de registros de cada lista respectivamente. Su valor cambia a medida que se vayan filtrando valores en el cuadro de búsqueda.

 

5. Seleccionar género.
Se selecciona un género entre Masculino, Femenino, o Ambos. El género por defecto es Ambos.

 

6. Seleccionar edad.
Se selecciona una edad entre Días o Años. La edad por defecto es Años. Si se selecciona días, la edad final sería 365 y si es años, 200.

 

7. Botón ayuda.
Aquí se despliega el manual de cada página.

 

8. Digitar rango de edades.
Se digitan los valores numéricos correspondientes al rango de años o días. Se valida si la edad inicial es mayor a la edad final. 

9. Listado de pruebas a asignar.
Es la lista de las pruebas que cuenta con las columnas de código, abreviatura, nombre y el check para asignar cada prueba.  Se puede filtrar y ordenar, por defecto aparecen ordenadas por la asignación. 

10. Asignar prueba.
Se marca o chequea una o varias pruebas a asignar.   

11.  Botón guardar.
Se guardan las pruebas marcadas para asignárselas al ítem demográfico seleccionado. 

12. Botón cancelar.
Se cancelan los cambios de asignación de las pruebas y la selección del ítem demográfico.

13. Botón reporte.
Se genera un reporte de la lista de las pruebas agrupadas por el ítem demográfico seleccionado. El reporte se puede exportar en formatos Excel o PDF. 

14. Listado de los items del demográfico configurado.
Es la lista de los ítems demográficos activos que se visualizan con el nombre del ítem. Dicha lista puede ser ordenada y/o filtrada por dicho campo. La fila seleccionada siempre se mostrará resaltada en azul claro y en ese momento aparecerán las pruebas asociadas al ítem seleccionado, estas pruebas se visualizarán en una lista con un campo que indica lo que está asignado. También se observa la información correspondiente a género, edad y rango de edades.
