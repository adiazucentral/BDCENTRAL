Tipo de Estudio

Un tipo de estudio hace referencia a la clasificación interna de los casos de patología permitiendo tener una mejor evaluación de los aspectos y diversos componentes que implican realizar el procesamiento y análisis de las muestras, ejemplos de tipos de estudio son quirúrgico, biología molecular, citometría de flujo, etc.

A continuación, se muestra una ventana típica con sus partes:

 

[Imagen 01 - Formulario de tipos de estudio]

 

1. Buscador.
Escribe una palabra o frase en el cuadro de búsqueda para buscar un elemento. Dado que la búsqueda se inicia en cuanto empiezas a escribir, si por ejemplo escribe "B", todos los tipos de estudio cuyo nombre empiece por la letra B se mostrarán en la lista.

2. Ordenamiento.
Usa el control de ordenamiento para ordenar de menor a mayor o viceversa según el campo que desees ordenar solo al hacer click  sobre él.  

3. Cantidad.
Este control muestra la cantidad de tipos de estudios visualizados en la lista.

4. Fecha - Usuario de última actualización.
Aquí se despliega la fecha y el usuario que modificaron el tipo de estudio que ha sido seleccionado.

5. Botón ayuda.
Aquí despliega la explicación de cada uno de los controles

6. Código.
Identificador único de los tipos de estudio, este campo solo acepta caracteres numéricos, lo máximo son 8 caracteres es un dato que siempre es requerido.

7. Nombre.
Nombre del tipo de estudio. Acepta un máximo de 16 caracteres es alfanumérico sin caracteres especiales y es requerido.

8. Estado.
Indica si el tipo de estudio se encuentra activo o inactivo. Cuando se crea un nuevo tipo de estudio este toma el valor de activo. Este campo es booleano y requerido.

9. Estudios.
Esta opción despliega la lista de estudios a los cuales se puede asociar al tipo de estudio. Esta opción es requerida, como mínimo debe existir un estudio asociado al tipo de estudio. Un estudio no puede estar asociado a dos tipos de estudio.

10. Botón nuevo.
Este botón habilita las cajas de texto (excepto el interruptor de estado) para crear una nuevo tipo de estudio. Si dichas cajas de texto tienen datos, se limpian. Al pulsarlo, se habilita el botón Cancelar y se deshabilitan los botones Editar y Reporte . Este botón por defecto está habilitado.

11. Botón editar.
Este botón permite editar el tipo de estudio seleccionado de la lista de datos y es habilitado cuando se selecciona un tipo de estudio de la lista. Se cargan los datos en las cajas de texto y al pulsarlo el botón Editar, se habilitan todos los controles del formulario. Se habilitan los botones Guardar y Cancelar, y se deshabilita el botón Reporte.

12. Botón guardar.
Este botón permite guardar un tipo de estudio ya sea creado o modificado. Al pulsarlo se deshabilitan los botones Editar, Cancelar y el propio botón, y se habilitan los botón Nuevo y Reporte.

13. Botón cancelar.
Este botón cancela, limpia y deshabilita las cajas de texto incluido el interruptor del estado. Deshabilita los botones Editar y Guardar, y habilita el botón Reporte.

14. Botón  Reporte.
Este botón imprime los tipos de esudio que se encuentran en la lista de datos. Se muestra un reporte con las mismas columnas y los mismos registros filtrados.

15. Listado de tipos de estudio.
Es una lista de datos que contiene tres campos: Código, Nombre y Estado. Dicha lista puede ser ordenada y/o filtrada por cada campo. La fila seleccionada siempre se mostrará resaltada en azul claro.
