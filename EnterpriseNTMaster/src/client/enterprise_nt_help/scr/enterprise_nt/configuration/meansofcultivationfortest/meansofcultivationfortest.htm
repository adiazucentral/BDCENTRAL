Medio de cultivo por prueba

Este maestro Asocia los medios de cultivo con las pruebas de microbiología que los emplearán

A continuación, se muestra una ventana típica con sus partes:

     

 

      1.  Ordenamiento.

Se usa el ordenamiento por columnas de menor a mayor o viceversa, la lista se organiza ya sea alfabéticamente, numéricamente. El ícono que aparece en cada columna cambia según el orden.

 

      2.  Cuadro de búsqueda.                                                                                           

Se escribe o se introduce un valor alfa-numérico para filtrar elementos de la lista de pruebas o medios de cultivo. El filtro se da de manera dinámica; a medida que se vaya digitando un valor, la lista va mostrando las filas coincidentes con las columnas visibles.

 

      3.  Cantidad. 

Este control muestra la cantidad de datos visualizados en la lista. Su valor cambia a medida que se vayan filtrando valores en el cuadro de búsqueda.

 

      4.  Usuario- fecha de última actualización.                                                                                                    

Aquí se muestra la fecha y el usuario que modificaron la prueba que ha sido  seleccionado. Por defecto no aparece visible, solo cuando se selecciona una  fila de la lista. 

 

                                                                                                                             

      5.  Botón ayuda.

Aquí se despliega el manual de cada página.

 

      6.   Lista de medios de cultivo.

Es una lista de datos que contiene cuatro campos: Código, Medio de cultivo, asignado, valor por defecto. Dicha lista puede ser ordenada y/o filtrada por cada campo. El campo asignado se selecciona cuando se quieren asignar los medios de cultivo a la  prueba que se seleccionó en la lista de pruebas y el valor por defecto son los elementos que se quieren mostrar al momento de cargar la lista de medios de cultivo.

 

      7.  Botón guardar.

Este botón permite guardar los medios de cultivo asociados. Al pulsarlo se deshabilitan los botón  Cancelar y el propio botón, y se habilitan el  botón  Imprimir.

 

      8.  Botón cancelar.

Este botón cancela, se devuelve al estado anterior antes de modificar el formulario. Deshabilita los botón Guardar, y habilita el botón Imprimir.

 

      9.  Botón reporte.

Este botón imprime una lista de medios de cultivo  que se encuentran asociadas y no asociadas a la prueba que selecciono.

 

      10.  Lista de pruebas.

Es una lista de datos que contiene dos campos: código y la prueba que contiene una muestra asociada a  microbiología. Dicha lista puede ser ordenada y/o filtrada por cada campo. La fila seleccionada siempre se mostrará resaltada en azul claro.
