Días de procesamiento

Este formulario permite en una forma ágil la definición de los días en que se procesa una prueba o perfil en el laboratorio.

A continuación, se muestra la ventana especial para la modificación de los días de procesamiento de una prueba:  

 

[Imagen de referencia - Pestaña de días de procesamiento.]

 

 

1. Ordenamiento.
Se usa el ordenamiento por columnas de menor a mayor o viceversa, la lista se organiza ya sea alfabéticamente o por valor (falso o verdadero) . El ícono que aparece en cada columna cambia según el orden.  

2. Cuadro de texto de búsqueda de la lista de áreas.
Se escribe o se introduce un valor alfa-numérico para filtrar elementos de la lista de datos. el filtro de se manera dinámica, a medida que se vaya digitando un valor, la lista va mostrando las filas coincidentes con las columnas visibles.  

3. Cantidad de datos en la lista de áreas.
Este control muestra la cantidad de áreas visualizados en las listas de datos de áreas. Su valor cambia a medida que se vayan filtrando valores en el cuadro de búsqueda.  

4. Fecha de última actualización-Usuario.
Aquí se muestra la fecha y el usuario que modificó el día de procesamiento de las pruebas del área seleccionada. Por defecto no aparece visible, solo cuando se selecciona una fila de la lista de áreas.       

5. Botón de ayuda
Aquí se despliega el manual de cada página. 

6. Cantidad de datos en la lista de pruebas.
Este control muestra la cantidad de pruebas visualizados en las listas de datos. Su valor cambia a medida que se vayan filtrando valores en el cuadro de búsqueda.      

7. Seleccionar todos.
Se tiene la posibilidad de pulsar el botón Todos para seleccionar todos los días de todas las pruebas o si lo prefiere se puede seleccionar un días para todas las pruebas visibles .  

8. Listado de pruebas.
Es donde se visualizan las pruebas con sus respectivos días de procesamiento. Se marcan los dias que desea aplicar y luego se guarda.   

 

9. Botón guardar.

Se guardan los cambios hechos en la lista de pruebas, para el caso son los días de procesamiento.

 

10. Botón cancelar.

Se cancelan los cambios hechos en la lista de pruebas.

 

11. Listado de áreas.
Es la lista de las áreas activas en el sistema y cuenta con los campos de Código y Nombre de área. Dicha lista puede ser ordenada y/o filtrada por cada campo. La fila seleccionada siempre se mostrará resaltada en azul claro y en ese momento aparecerán las pruebas asociadas al árera seleccionada, dichas pruebas se visualizarán en una lista con sus respectuvos días de procesamiento.
