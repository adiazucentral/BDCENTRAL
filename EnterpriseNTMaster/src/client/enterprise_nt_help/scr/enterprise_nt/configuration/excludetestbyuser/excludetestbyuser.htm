Excluir prueba por usuario

Este maestro permite seleccionar las pruebas que no se van  a procesar por cada uno de los usuarios

A continuación, se muestra una ventana típica con sus partes:

    

      1.Ordenamiento.

Se usa el ordenamiento por columnas de menor a mayor o viceversa, la lista se organiza ya sea alfabéticamente, numéricamente. El ícono que aparece en cada columna cambia según el orden.

 

      2.Cuadro de búsqueda.                                                                                          

Se escribe o se introduce un valor alfa-numérico para filtrar elementos de la lista de usuarios o la pruebas. El filtro se da de manera dinámica; a medida que se vaya digitando un valor, la lista va mostrando las filas coincidentes con las columnas visibles. 

 

      3.Cantidad. 

Este control muestra la cantidad de datos visualizados en la lista. Su valor cambia a medida que se vayan filtrando valores en el cuadro de búsqueda.

 

      4.Usuario- fecha de última actualización.                                                                                                     

Aquí se muestra la fecha y el usuario que modificaron el usuario   que ha sido  seleccionado. Por defecto no aparece visible, solo cuando se selecciona una  fila de la lista.

 

      5.Botón ayuda.

Aquí se despliega el manual de cada página.

 

      6.Lista de Pruebas.

Es una lista de datos que contiene cuatro campos: Código, Abreviatura, Prueba y asignado. Dicha lista puede ser ordenada y/o filtrada por cada campo. El campo asignado se selecciona cuando se quieren excluir las pruebas al usuario que se seleccionó en la lista de usuarios.

 

      7. Botón guardar.

Este botón permite guardar las pruebas excluidas. Al pulsarlo se deshabilitan los botón  Cancelar y el propio botón, y se habilitan el  botón  Imprimir.

 

      8. Botón cancelar.

Este botón cancela, se devuelve al estado anterior antes de modificar el formulario. Deshabilita los botón Guardar, y habilita el botón Imprimir.

 

9. Botón reporte.

Este botón imprime una lista de pruebas que se encuentran asociadas y no asociadas al usuario seleccionado.

     

      10. Lista de Usuarios.

Es una lista de datos que contiene dos campos: Usuario y Nombre. Dicha lista puede ser ordenada y/o filtrada por cada campo. La fila seleccionada siempre se mostrará resaltada en azul claro.
