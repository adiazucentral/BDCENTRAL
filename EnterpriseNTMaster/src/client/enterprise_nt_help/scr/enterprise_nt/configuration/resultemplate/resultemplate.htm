Plantilla de resultado

El maestro de plantilla de resultado define una lista de selección multiple tipo texto para el ingreso de un resultado

A continuación, se muestra una ventana típica con sus partes:

 

1. Ordenamiento.

Se usa el ordenamiento por columnas de menor a mayor o viceversa,la lista se organiza ya sea alfabéticamente, numéricamente. El ícono que aparece en cada columna cambia según el orden.

 

2.    2.Cuadro de búsqueda.                                                                                          

Se escribe o se introduce un valor alfa-numérico para filtrar elementos de la lista de pruebas. El filtro se da de manera dinámica; a medida que se vaya digitando un valor, la lista va mostrando las filas coincidentes con las columnas visibles.

 

3. Area .

Area por la que se desea filtrar la prueba.  

 

4. Cantidad .

Este control muestra la cantidad de pruebas visualizados en la lista de datos. Su valor cambia a medida que se vayan filtrando valores en el cuadro de búsqueda.

 

4.     5. Usuario - fecha de última actualización.                                                                                                      

Aquí se muestra la fecha y el usuario que modificaron el valor del de la plantilla de resultado  que ha sido  seleccionado. Por defecto no aparece visible, solo cuando se selecciona una fila de la lista de pruebas .

 

5.   6. Botón ayuda.

Aquí se depliega el manual de cada pagina.

 

6.     7. Botón de eliminación.

      Este botón elimina la opción de la plantilla de resultados.

 

8. Opción.

Campo que representa las  preguntas que se le realizara a cada prueba.

 

9. Comentario.

Texto que complementa las respuestas de la opción.

 

10. Agregar resultado.

Botón que agrega un nuevo resultado .

 

11. Eliminar resultado.

Boton que elimina un resultado.

 

12. Listado resultado.

Lista de posibles valores que puede tener cada una de las preguntas formuladas. dato en blanco no se almacena. 

 

13.  Botón nuevo. 

Este botón crea una plantilla de resultados y habilita las cajas de texto. si dicha cajas de texto tienen datos se limpian. Al pulsarlo, se habilita el botón Cancelar y se deshabilitan los botones  Editar y Reporte.   

 

14.  Botón guardar.

Este botón permite guardar una plantilla de resultado . Al pulsarlo se deshabilitan los botones Editar, Cancelar y el propio botón, y se habilita el boton  Reporte.

 

15. Botón cancelar.

Este boton cancela, limpia y deshabilita las cajas de texto.  Deshabilita los botones Editar y Guardar,y habilita el botón Reporte.

 

16. Botón Reporte .

Este botón imprime las plantillas de resultado asociadas a la prueba que se encuentra en la lista de datos.

 

17. Lista de pruebas .

Lista de datos que contiene dos campos: Código, abreviatura y nombre de la prueba . Dicha lista puede ser ordenada y/o filtrada por cada campo. La fila seleccionada siempre se mostrará re saltada en azul claro.
