Médicos

El maestro define los médicos que remiten a los pacientes al laboratorio

A continuación, se muestra una ventana típica con sus partes:

1. Cuadro de búsqueda.                                                                                           

Se escribe o se introduce un valor alfa-numérico para filtrar elementos de la lista de datos de médicos. El filtro se da de manera dinámica; a medida que se vaya digitando un valor, la lista va mostrando las filas coincidentes con las columnas visibles. 

2. Ordenamiento.  

Se usa el ordenamiento por columnas de menor a mayor o viceversa, la lista se organiza ya sea alfabéticamente, numéricamente. El ícono que aparece en cada columna cambia según el orden. 

3. Cantidad.    

Este control muestra la cantidad de médicos visualizados en la lista de datos. Su valor cambia a medida que se vayan filtrando valores en el cuadro de búsqueda o cuando se introduce un nuevo médico.         

4. Usuario- fecha de última actualización.                                                                                                      

Aquí se muestra la fecha y el usuario que modificaron el médico que ha sido  seleccionado. Por defecto no aparece visible, solo cuando se selecciona una  fila de la lista de médicos. 

5. Botón ayuda.  

Aquí se despliega el manual de cada página.

6. Identificación.

Número de identificación único para el médico, el cual no se puede repetir. Es un campo alfa-numérico que acepta mayúsculas y minúsculas, lo máximo son 60 caracteres, Es un campo requerido.

7. Código.

Identificador único del médico, el cúal no se puede repetir, De preferencia nemotécnico. Es un campo numérico;  es un dato que siempre será requerido.

8. Especialidad.

Este campo indica la Especialidad del médico que solicita los análisis.

9. Nombre.

Este campo Indica el Nombre(s) del médico que remite las ordenes, Es un campo alfa-numérico que acepta mayúsculas y minúsculas, lo máximo son 60 caracteres, Es un campo requerido.

10. Apellido.

Este campo Indica el Apellido(s) del médico que remite las ordenes, Es un campo alfa-numérico que acepta mayúsculas y minúsculas, lo máximo son 60 caracteres, Es un campo requerido.

11. Correo.

Este campo indica el correo para enviar resultados al médico al momento de imprimir las ordenes, Es un campo alfa-numérico que acepta mayúsculas y minúsculas, lo máximo son 60 caracteres. 

12. Dirección 1.

Este Campo indica la dirección del médico, Es un campo alfa-numérico que acepta mayúsculas y minúsculas, lo máximo son 60 caracteres.

13. Dirección 2.

Este campo indica una dirección adicional del médico, Es un campo alfa-numérico que acepta mayúsculas y minúsculas, lo máximo son 60 caracteres.

14. Código postal.

Este campo indica el número que corresponde a la zona postal, Es un campo alfa-numérico que acepta mayúsculas y minúsculas, lo máximo son 10 caracteres.

15. Ciudad.

Este campo Indica la ciudad donde se encuentra el médico ubicado o donde se le puede enviar la correspondencia., Es un campo alfa-numérico que acepta mayúsculas y minúsculas, lo máximo son 60 caracteres.

16. Departamento.

Este campo Indica el departamento donde se encuentra ubicado el médico  o donde se le puede enviar la correspondencia., Es un campo alfa-numérico que acepta mayúsculas y minúsculas, lo máximo son 60 caracteres.

17. Teléfono.

Número de contacto del médico . Posee una máscara que es configurable en configuración.

18. Fax.

Este campo indica el número de fax para el envío de documentos, Es un campo alfa-numérico que acepta mayúsculas y minúsculas, lo máximo son 60 caracteres.

19. Copia informe resultados.

Este campo indica si se le envía copia del informe de resultados al médico a su correo electrónico, esta acción se haría desde el módulo de resultados en el momento de la impresión cuando se hayan validado todos los exámenes. Este campo se puede  diligenciar sólo si se ingresó correo electrónico.

20. Estado.

Este botón Indica si registro está activo o no, al modificar el estado siempre hay aparece un cuadro de dialogo que pide la confirmación.

21. Correos adicionales.

Este campo Indica correos de medicos adicionales para enviar los informes .

21. ID NPI.

Este campo Indica el número del national provider identification, Es un campo alfa-numérico que acepta mayúsculas y minúsculas, lo máximo son 80 caracteres. Solo se encontrara activo si la facturación de Billings USA esta activa.

22. ID Licencia médico.

Este campo indica el número del national provider identification (Identificación del proveedor Nacional). Es un campo alfa-numérico que acepta mayúsculas y minúsculas, lo máximo son 80 caracteres, Es un campo requerido. Solo se encontrara activo si la facturación de Billings USA esta activa.

23. Identificación entidad.

Este campo Indica  la Identificación de la entidad a la que se factura., Es un campo alfa-numérico que acepta mayúsculas y minúsculas, lo máximo son 80 caracteres.  Solo se encontrara activo si la facturación de Billings USA esta activa.

24. Botón nuevo.

Este botón habilita las cajas de texto (excepto el interruptor de estado) para crear una nuevo médico. Si dichas cajas de texto tienen datos, se limpian.

Al pulsarlo, se habilita el botón Cancelar y se deshabilitan los botones Editar e Imprimir. Este botón por defecto debe estar habilitado.

25. Botón editar.

Este botón permite editar el médico seleccionado de la lista de datos y es habilitado cuando se selecciona un médico de la lista. Se cargan los datos en las cajas de texto y al pulsarlo el botón Editar, se habilitan todos los controles del formulario. Se habilitan los botones Guardar y Cancelar, y se deshabilita el botón Imprimir.

26. Botón guardar.

Este botón permite guardar un médico ya sea creada o modificada. Al pulsarlo se deshabilitan los botones Editar, Cancelar y el propio botón, y se habilitan los botones Nuevo e Imprimir.

27. Botón cancelar.

Este botón cancela, limpia y deshabilita las cajas de texto incluido el interruptor del estado. Deshabilita los botones Editar y Guardar, y habilita el botón Imprimir.

28. Botón reporte

Este botón imprime los médicos que se encuentran en la lista de datos. Se muestra un reporte con las mismas columnas y los mismos registros filtrados.

29. Lista de Médico.

Es una lista de datos que contiene tres campos: Nombre, código, Especialidad y Estado. Dicha lista puede ser ordenada y/o filtrada por cada campo. La fila seleccionada siempre se mostrará resaltada en azul claro y las muestras inactivas estarán seleccionadas con rosado.
