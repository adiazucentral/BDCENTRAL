Valores por demográfico

Este maestro define los valores por defecto para los demograficos que son obligatorios, para cuando los demográficos esten configurado por sede y no se encuentre configurado.

A continuación, se muestra una ventana típica con sus partes:

 

     

      1. Ordenamiento.

Se usa el ordenamiento por columnas de menor a mayor o viceversa, la lista se organiza ya sea alfabéticamente, numéricamente. El ícono que aparece en cada columna cambia según el orden.

 

      2. Cuadro de búsqueda.                                                                                          

Se escribe o se introduce un valor alfa-numérico para filtrar elementos de la lista demográficos. El filtro se da de manera dinámica; a medida que se vaya digitando un valor, la lista va mostrando las filas coincidentes con las columnas visibles.

 

3. Botón ayuda.

Aquí se despliega el manual de cada página.

 

4. Cantidad. 

Este control muestra la cantidad demográficos visualizados en la lista de datos. Su valor cambia a medida que se vayan filtrando valores en el cuadro de búsqueda .

 

       5. D emográficos.

Es una lista de datos que contiene nombre de los demograficos que estan configurados obligatorios, dicha lista puede ser ordenada y/o filtrada por cada campo.

 

6. Valor por defecto .

Es una lista de los items demográficos para seleccionar uno por defecto.

 

       7. Botón guardar.

Este botón permite guardar  los cambios.
