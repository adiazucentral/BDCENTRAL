Medio de cultivo

Este maestro define los medios de cultivo para la siembra de microbiología.

A continuación, se muestra una ventana típica con sus partes:

     

      1. Ordenamiento.

Se usa el ordenamiento por columnas de menor a mayor o viceversa, la lista se organiza ya sea alfabéticamente, numéricamente. El ícono que aparece en cada columna cambia según el orden.

      2. Cuadro de búsqueda.                                                                                          

Se escribe o se introduce un valor alfa-numérico para filtrar elementos de la lista de medios de cultivo. El filtro se da de manera dinámica; a medida que se vaya digitando un valor, la lista va mostrando las filas coincidentes con las columnas visibles.

3. Cantidad. 

Este control muestra la cantidad de  medios de cultivo visualizados en la lista de datos. Su valor cambia a medida que se vayan filtrando valores en el cuadro de búsqueda o cuando se introduce un nuevo  medio de cultivo.

      4. Usuario- fecha de última actualización.                                                                                                    

Aquí se muestra la fecha y el usuario que modificaron el  medio de cultivo que ha sido  seleccionado. Por defecto no aparece visible, solo cuando se selecciona una  fila de la lista de  medios de cultivo.

5. Botón ayuda.

Aquí se despliega el manual de cada página.

6. Código.

Código que identifica el medio de cultivo  el cual no se puede repetir. Es un campo alfa-numérico que acepta mayúsculas y minúsculas, lo máximo son 10 caracteres, Es un campo requerido.  

      7. Medio de cultivo.

Nombre del  medios de cultivo  a definir el cual no se puede repetir. Es un campo alfa-numérico que acepta mayúsculas y minúsculas, lo máximo son 60 caracteres, Es un campo requerido.

      8. Estado.

Este botón Indica si registro está activo o no, al modificar el estado siempre hay aparece un cuadro de dialogo que pide la confirmación.

      9. Botón nuevo.

Este botón habilita las cajas de texto (excepto el interruptor de estado) para crear un nuevo  medio de cultivo. Si dichas cajas de texto tienen datos, se limpian. Al pulsarlo, se habilita el botón Cancelar y se deshabilitan los botones Editar e Imprimir. Este botón por defecto debe estar habilitado.

      10. Botón editar.

Este botón permite editar el  medio de cultivo seleccionado de la lista de datos y es habilitado cuando se selecciona un  medio de cultivo de la lista. Se cargan los datos en las cajas de texto y al pulsarlo el botón Editar, se habilitan todos los controles del formulario. Se habilitan los botones Guardar y Cancelar, y se deshabilita el botón Imprimir.

      11. Botón guardar.

Este botón permite guardar un medio de cultivo ya sea creada o modificada. Al pulsarlo se deshabilitan los botones Editar, Cancelar y el propio botón, y se habilitan los botones Nuevo e Imprimir.

      12. Botón cancelar.

Este botón cancela, limpia y deshabilita las cajas de texto incluido el interruptor del estado. Deshabilita los botones Editar y Guardar, y habilita el botón Imprimir.

      13. Botón reporte.

Este botón imprime los  medios de cultivo que se encuentran en la lista de datos. Se muestra un reporte con las mismas columnas y los mismos registros filtrados.

      14. Lista de medios de cultivo.

Es una lista de datos que contiene dos campos: Código, medio de cultivo y Estado. Dicha lista puede ser ordenada y/o filtrada por cada campo. La fila seleccionada siempre se mostrará resaltada en azul claro y los  medios de cultivo inactivos estarán resaltados con rosado.
