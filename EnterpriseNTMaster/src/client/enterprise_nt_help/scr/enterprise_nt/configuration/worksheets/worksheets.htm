Hojas de trabajo

Las hojas de trabajo genera como su nombre lo indica una hoja de trabajo para el ingreso de resultados manuales. Este formulario permite agrupar una serie de pruebas manuales, para ser mostrados en un reporte.

A continuación, se muestra una ventana típica con sus partes:

      1. Cuadro de texto de búsqueda.

Se escribe o se introduce un valor alfa-numérico para filtrar elementos de la lista de datos de las hojas de trabajo. El filtro se da de manera dinámica; a medida que se vaya digitando un valor, la lista va mostrando las filas coincidentes con las columnas visibles.

 

      2. Ordenamiento.

Se usa el ordenamiento por columnas de menor a mayor o viceversa, la lista se organiza ya sea alfabéticamente, numéricamente o por valor (falso o verdadero). El ícono que aparece en cada columna cambia según el orden.

 

      3.   Cantidad. 

Este control muestra la cantidad de registros visualizados en la lista de datos. Su valor cambia a medida que se vayan filtrando valores en el cuadro de búsqueda o cuando se introduce una nueva hoja de trabajo.

 

      4.   Fecha de última actualización - Usuario.

Aquí se muestra la fecha y el usuario que creó o modificaron el registro que ha sido seleccionado. Por defecto no aparece visible, solo cuando se selecciona una fila de la lista de las hojas de trabajo.

 

      5.   Botón ayuda. 

Aquí se despliega el manual de cada página.

 

 6.  Nombre.

Nombre de la hoja de trabajo

   

7. Microbiología.

Indica si genera el reporte de microbiología o no. Este reporte no aplica orientación, tipo o si es excluyente al momento de generar la hoja de trabajo

 

8. Tipo de hoja .

Indica si la hoja de trabajo es una hoja de trabajo o un listado

 

9. Orientacion .

Indica si la hoja de trabajo es vertical u horizontal. Adicionalmente si es listado solo se puede vertical.

 

10.  Excluyente.

Indica que todos los examenes de la hoja de trabajo deben estar dentro de las ordenes.

 

11.  Estado.

Indica si la hoja de trabajo se encuentra activa o no. Al cambiar el estado muestra un cuadro de confirmación donde pregunta si se encuentra seguro de cambiar el estado o no. El control tiene forma de interruptor (switch).

 

12. Pruebas.

Listado de pruebas que contiene la hoja de trabajo. La cantidad depende de la orientación y el tipo.

 

13.   Botón nuevo.

Este botón habilita los controles (excepto el interruptor de estado) para crear una nueva hoja de trabajo. Si dichos controles tienen información, se limpian.

Al pulsarlo, se habilita el botón Cancelar y se deshabilitan los botones Editar y Reporte. Este botón por defecto debe estar habilitado.

 

      14.Botón editar.

Este botón permite editar la relacion seleccionada de la lista de datos y esta habilitado cuando se selecciona un registro de la lista. Se cargan los datos en la caja de texto y al pulsarlo el botón Editar, se habilitan todos los controles del formulario. Se habilitan los botones Guardar y Cancelar, y se deshabilita el botón Reporte.

 

      15.  Botón guardar.

Este botón permite guardar una hoja de trabajo ya sea creada o modificada. Al pulsarlo se deshabilitan los botones Editar, Cancelar y el propio botón, y se habilitan los botones Nuevo e  Reporte.

 

      16.  Botón cancelar.

Este botón cancela, limpia y deshabilita los controles del formulario incluido el interruptor del estado. Deshabilita los botones Editar y Guardar, y habilita el botón Reporte.

 

      17.  Botón reporte

Este botón abre una ventana donde se muestra una previsualizacion del reporte, en este se podra escoger en que formato se desea exportar el listado de hojas de trabajo, este informe debe ser acorde al filtro y ordenamiento escogido en la tabla de datos del formulario inicial.

 

      18.  Listado de hojas de trabajo.

Es una lista de datos que contiene los campos de: nombre y estado. Dicha lista puede ser ordenada y/o filtrada por cada campo. La fila seleccionada siempre se mostrará resaltada en azul claro. La fila inactiva se mostrará en rosado.
